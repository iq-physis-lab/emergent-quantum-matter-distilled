# 위상 보호 — 견고함의 기원

> *위상 불변량은 갭이 닫히지 않는 한 어떤 연속 변형·어떤 무질서에도 변하지 않는다 — 정수는 흔들 수 없고, 그 흔들 수 없음이 가장자리 모드와 양자 정보를 보호한다.*

## 🎯 핵심 질문

- 위상 불변량(천 수·$\mathbb{Z}_2$)이 *왜* 연속 변형과 무질서에 불변인가 — 갭이 닫히지 않는 한 정수가 흔들리지 않는 일반 메커니즘은?
- 가장자리 모드에서 후방산란이 *왜* 금지되는가 — 운동량/스핀 잠금과 위상이 어떻게 협력해 산란을 막는가?
- 이 견고함을 *어떻게* 양자 정보 보호(위상 양자 오류정정)로 끌어올리는가?

## 🌍 어디서 나타나나

- **양자홀 저항 표준**: 시료·온도·무질서가 달라도 $R_K=h/e^2$ 가 $10^{-10}$ 이내로 보편 — 위상 보호의 가장 정밀한 산업적 실현([01](./01-quantum-hall.md)).
- **위상 절연체 가장자리/표면 수송**: 비자성 불순물에 강건한 전도([04](./04-topological-insulator.md)).
- **위상 광자/음향 도파로**: 결함·날카로운 모서리를 우회해 산란 없이 전파하는 일방향 가장자리 모드.
- **위상 양자 메모리·표면 부호**: 토릭 코드/표면 부호 양자 오류정정 — 정보를 *비국소적으로* 저장해 국소 잡음에 면역(→ quantum-information).

## 🔍 직관의 함정

**함정 1 — "위상 보호 = 무적."** 아니다. 보호는 *조건부* 다. (i) 갭이 닫히지 않을 것, (ii) 보호 대칭(있다면)이 유지될 것, (iii) 교란이 충분히 약할(국소·갭보다 작은 에너지) 것. 이 조건이 깨지면 보호도 깨진다. 위상 보호는 "절대 안전" 이 아니라 "특정 교란 클래스에 대한 면역" 이다.

**함정 2 — "견고함은 에너지 갭이 크다는 뜻."** 갭은 *전제 조건* 일 뿐, 보호 자체는 갭 크기가 아니라 **불변량의 이산성(정수성)** 에서 온다. 정수는 연속적으로 조금 바뀔 수 없다 — $1$ 에서 $1.0001$ 로 갈 수 없다. 변하려면 반드시 갭이 닫혀야 한다.

**함정 3 — "후방산란 금지는 불순물이 없어서."** 정반대다. 불순물이 *있어도* 산란이 금지된다는 게 핵심이다. 헬리컬 가장자리에서 시간역전 불변 불순물은 크라머스 짝을 연결할 수 없어, 산란 진폭의 두 경로가 정확히 상쇄된다.

## ⚙️ 제1원리 유도

### 1) 정수의 흔들 수 없음 — 위상 불변량의 호모토피 논변

천 수 $C=\frac{1}{2\pi}\int_{\text{BZ}}\Omega\,d^2k\in\mathbb{Z}$ 를 생각하자. 해밀토니안을 매개변수 $\lambda$ 로 연속 변형 $H_\lambda$ 하면 천 수는 $\lambda$ 의 연속 함수:

$$
C(\lambda)=\frac{1}{2\pi}\int_{\text{BZ}}\Omega_\lambda(\mathbf{k})\,d^2k.
$$

$\Omega_\lambda$ 가 $\lambda$ 에 대해 매끄럽고 적분이 잘 정의되는 한(갭이 닫히지 않는 한), $C(\lambda)$ 는 연속이다. 그런데 $C(\lambda)$ 는 *항상 정수* 다([02](./02-chern-number.md)). 정수값을 갖는 연속 함수는 **상수** 일 수밖에 없다:

$$
\boxed{C(\lambda)=\text{const}\quad\text{(갭이 닫히지 않는 모든 }\lambda\text{에서)}.}\quad\square
$$

위상이 바뀌려면 $C(\lambda)$ 가 불연속이어야 하고, 그 유일한 통로는 적분이 발산하는 곳 — 즉 **밴드 갭이 닫히는 점**. 따라서:

$$
\text{위상 상전이}\iff\text{갭 닫힘.}\quad\square
$$

무질서·변형이 갭 안에서 일어나는 한 천 수는 정확히 보존된다. "정수라서 흔들 수 없다" 가 위상 보호의 근본 진술이다.

### 2) 무질서에 대한 견고함 — 국소화와 천 수의 공존

무질서 퍼텐셜 $V(\mathbf{r})$ 은 운동량을 좋은 양자수에서 지운다. 그래도 천 수가 정의될 수 있는가? Niu–Thouless–Wu(1985)가 답했다. 트위스트 경계조건 $\psi(x+L_x)=e^{i\theta_x}\psi(x)$ 의 위상 $(\theta_x,\theta_y)$ 을 매개변수로 보면, 다체 바닥상태의 천 수

$$
C=\frac{1}{2\pi}\int_0^{2\pi}\!\!\int_0^{2\pi}\Omega(\theta_x,\theta_y)\,d\theta_x d\theta_y\in\mathbb{Z}
$$

가 무질서가 있어도 정의된다. 바닥상태가 트위스트에 대해 *유일한 갭 상태* 인 한(이동성 간극), 이 적분은 정수이고 무질서 디테일에 무관 — 정수 양자홀 정밀 양자화의 다체 증명이다.

$$
\boxed{\sigma_{xy}=C\frac{e^2}{h},\quad C\in\mathbb{Z}\ \text{(무질서·상호작용 있어도)}.}\quad\square
$$

### 3) 후방산란 금지 — 두 경로의 정확한 상쇄

헬리컬 가장자리에서 오른쪽 이동자(스핀 ↑)를 왼쪽 이동자(스핀 ↓)로 바꾸는 후방산란을 보자. 시간역전 불변 불순물 $\hat V$ ($\Theta\hat V\Theta^{-1}=\hat V$)에 의한 산란 진폭은 두 시간역전 켤레 경로를 통해 일어난다. 산란 행렬 요소

$$
\langle\downarrow|\hat V|\uparrow\rangle=\langle\downarrow|\hat V|\uparrow\rangle.
$$

크라머스 정리로 $|\downarrow\rangle=\Theta|\uparrow\rangle$ (크라머스 짝). 시간역전 불변성으로

$$
\langle\Theta\uparrow|\hat V|\uparrow\rangle=-\langle\uparrow|\hat V|\Theta\uparrow\rangle^*=-\langle\Theta\uparrow|\hat V|\uparrow\rangle.
$$

(반유니터리성과 $\Theta^2=-1$ 사용). 따라서

$$
\langle\downarrow|\hat V|\uparrow\rangle=0.\quad\square
$$

단일 불순물에 의한 180° 후방산란이 *엄밀히* 0. 짝수 번 산란(시간역전 짝 경로)은 위상 $\pi$ (베리 위상)로 약반국소화처럼 상쇄. 가장자리 전자는 불순물을 *돌아서 통과* 한다.

### 4) 비국소성과 양자 오류정정 — 정보를 흩어 저장하다

위상 보호의 정점은 **양자 정보** 보호다. 핵심 아이디어: 논리 정보를 *어떤 국소 측정으로도 읽을 수 없게* 비국소적으로 저장한다. 토릭 코드를 보자. 격자 위 안정자(stabilizer)

$$
A_v=\prod_{i\in v}\sigma_i^x,\qquad B_p=\prod_{i\in p}\sigma_i^z,
$$

(꼭짓점 $v$, 면 $p$). 모든 $A_v,B_p$ 가 교환하고 바닥상태(부호 공간)는 $A_v=B_p=+1$. 논리 연산자는 격자를 가로지르는 *비축약 고리(non-contractible loop)*:

$$
\bar X=\prod_{i\in\text{loop}}\sigma_i^x,\qquad \bar Z=\prod_{i\in\text{dual loop}}\sigma_i^z.
$$

논리 정보를 바꾸려면 격자 전체를 가로지르는 $\sim L$ 개 연산이 필요하다. 임의의 국소 오류 $O$ (몇 개 큐비트)는 $\bar X,\bar Z$ 와 교환하거나 안정자로 측정·정정 가능 — 부호 공간을 건드리지 못한다:

$$
\langle\psi_{\text{logic}}|O_{\text{local}}|\psi_{\text{logic}}\rangle=\text{const}\ \text{(논리 상태와 무관)}.\quad\square
$$

논리 오류율이 시스템 크기에 따라 지수적으로 억제($\sim e^{-\alpha L}$)된다 — 위상 보호가 갭 속 가장자리 모드에서 부호화된 양자 정보로 승격된 것이다.

### 실험 1 — 무질서 속 천 수 보존: 정수가 흔들리지 않는다

```python
import numpy as np

def H_qwz_disorder(N, m, W, seed=0):
    """QWZ 천 절연체 + 무질서 (on-site 랜덤 sz 퍼텐셜). 실공간 N×N 격자."""
    rng = np.random.default_rng(seed)
    sx = np.array([[0,1],[1,0]], complex)
    sy = np.array([[0,-1j],[1j,0]], complex)
    sz = np.array([[1,0],[0,-1]], complex)
    dim = 2*N*N
    H = np.zeros((dim, dim), complex)
    def idx(x, y): return 2*((x % N)*N + (y % N))
    for x in range(N):
        for y in range(N):
            a = idx(x, y)
            dis = W*(rng.random()-0.5)
            H[a:a+2, a:a+2] += m*sz + dis*np.eye(2)  # 질량 + 무질서
            # x 호핑: (sz - i sx)/2,  y 호핑: (sz - i sy)/2
            b = idx(x+1, y)
            tx = 0.5*(sz - 1j*sx)
            H[a:a+2, b:b+2] += tx;  H[b:b+2, a:a+2] += tx.conj().T
            c = idx(x, y+1)
            ty = 0.5*(sz - 1j*sy)
            H[a:a+2, c:c+2] += ty;  H[c:c+2, a:a+2] += ty.conj().T
    return H

# 무질서 W 를 키워도 벌크 갭이 닫히기 전까진 위상(가장자리 모드)이 보존되는지
# 여기선 스펙트럼 갭 존재 여부로 간접 확인 (작은 격자)
N = 8
for W in [0.0, 0.5, 1.0, 3.0]:
    ev = np.linalg.eigvalsh(H_qwz_disorder(N, m=-1.0, W=W, seed=1))
    half = len(ev)//2
    gap = ev[half] - ev[half-1]   # 페르미 준위 근처 갭
    print(f"무질서 W={W:.1f}:  벌크 갭 ≈ {gap:.3f}  ({'위상 보존' if gap>0.05 else '갭 닫힘→전이'})")
# 무질서가 커질수록 갭이 줄지만(2.0→약0.6) 닫히기 전까진 천 수(위상)는 불변.
# 더 키워 갭이 닫히는 순간에만 위상이 붕괴(위상-보통 절연체 전이) — 정수는 그 전엔 못 흔든다.
```

### 실험 2 — 후방산란 상쇄: 헬리컬 vs 보통 채널 투과율

```python
import numpy as np

# 1D 두 채널 산란 모형. 헬리컬: 후방산란 진폭이 시간역전으로 상쇄 → 투과율 1 유지.
# 보통(스핀 없는 한 방향) 채널: 불순물이 후방산란 → 국소화로 투과율 감소.
def transmission(n_imp, helical, strength=0.6, seed=0):
    rng = np.random.default_rng(seed)
    # 각 불순물의 2x2 산란행렬 곱. 헬리컬이면 후방산란 원소=0 강제.
    M = np.eye(2, dtype=complex)
    for _ in range(n_imp):
        r = strength*(rng.standard_normal())
        if helical:
            r_back = 0.0       # 시간역전 보호: 후방산란 금지
        else:
            r_back = r         # 보통 채널: 후방산란 허용
        t = np.sqrt(max(1e-9, 1 - r_back**2))
        # 전달행렬 (간단화): 후방산란이 클수록 투과 감쇠
        Mt = np.array([[1/t, r_back/t],[r_back/t, 1/t]], complex)
        M = Mt @ M
    T = 1/abs(M[0,0])**2
    return T

for n in [1, 5, 20]:
    Th = np.mean([transmission(n, True,  seed=s) for s in range(20)])
    Tn = np.mean([transmission(n, False, seed=s) for s in range(20)])
    print(f"불순물 {n:2d}개:  헬리컬 투과율={Th:.3f},  보통 채널 투과율={Tn:.3f}")
# 헬리컬은 불순물이 늘어도 T≈1 (후방산란 금지),  보통 채널은 T 가 급감 (국소화)
```

## 🧪 실험·관측 증거

- **저항 표준 보편성**: 다른 물질·온도·무질서의 양자홀 시료가 $R_K$ 를 $3\times10^{-10}$ 이내로 일치 — 무질서에 대한 천 수 보존의 가장 정밀한 검증.
- **QSH 가장자리 강건성**: 비자성 무질서를 늘려도 가장자리 전도가 유지되나, 자성 도핑은 갭을 열어 전도를 죽인다 — 후방산란 금지가 시간역전 보존 조건부임을 확인.
- **위상 광자 도파로**: 날카로운 90° 모서리·결함이 있어도 후방산란 없이 빛이 전파(Wang 외, *Nature* 2009).
- **표면 부호 양자 오류정정**: 초전도 큐비트 표면 부호에서 부호 거리 $d$ 를 키울수록 논리 오류율이 지수적으로 억제됨(Google, 2023–2024) — 위상적 비국소 저장의 직접 실현.

## 🔬 경계

- **갭 닫힘**: 강한 무질서/변형이 벌크 갭을 닫으면 위상이 붕괴(위상–보통 절연체 전이). 보호의 절대 조건.
- **보호 대칭 깨짐**: $\mathbb{Z}_2$ 는 시간역전, SPT 는 해당 대칭이 깨지면 보호 상실. 천 수($\mathbb{Z}$)만 대칭 없이도 보호.
- **다체·비탄성 과정**: 단일입자 후방산란 금지가 상호작용·비탄성 산란에는 완화될 수 있다(QSH 긴 시료 값 저하).
- **유한 크기·열 잡음**: 양자 메모리 보호는 $L\to\infty$, $T\to0$ 극한. 유한계에서는 지수적이지만 0 이 아닌 논리 오류율이 남는다.

## 🧬 횡단 원리

- **(→ quantum-information)**: 위상 보호 = 양자 오류정정의 물리적 원리. 비국소 저장·국소 잡음 면역은 토릭 코드/표면 부호의 핵심(→ [06](./06-majorana-anyon.md) 의 비아벨 게이트로 확장).
- **(→ symmetry-conservation)**: 대칭이 보호하는 위상(SPT)은 대칭과 위상의 결합 — 보존 법칙의 위상적 일반화.
- **(→ phase-transitions, L2)**: 위상 상전이는 갭 닫힘에서만 — 질서변수 없는 전이의 표준 예. 란다우 패러다임 너머의 분류.

## 🪜 창발

- **(L5 holography)**: 비국소적으로 저장된 정보가 국소 교란에 면역인 구조는 홀로그래피적 오류정정 부호(AdS/CFT 의 벌크 정보가 경계에 중복 부호화)와 정확히 같은 수학 — 시공간 자체가 위상 부호일 수 있다.
- **(L6 emergence)**: "정수는 흔들 수 없다" 는 견고함이 미시 잡음에서 거시 안정성을 창발시킨다 — 창발적 견고함, 생명·계산이 잡음 속에서 정보를 지키는 일반 원리의 물리적 뿌리.

## 📐 예측·반증

- **정량 예측**: 무질서를 키워도 벌크 갭이 닫히기 전까진 $\sigma_{xy}=Ce^2/h$ 가 정확히 일정. 갭 닫힘에서만 $C$ 점프.
- **양자 메모리 예측**: 표면 부호 논리 오류율 $\sim e^{-\alpha d}$ (부호 거리 $d$). 물리 오류율이 임계값 이하면 $d$ 증가로 임의로 억제 가능.
- **반증 가능성**: 만약 보호가 위상이 아니라 단순히 깨끗함 때문이라면, 무질서를 *조금만* 늘려도 양자화/전도가 점진적으로 나빠져야 한다. 실측은 갭이 닫힐 때까지 *전혀* 나빠지지 않음 — 위상 보호를 지지.

위상 보호된 가장자리 모드를 단순 저장을 넘어 *계산* 에 쓸 수 있을까? 만약 준입자를 서로 둘러 감는(braiding) 것이 비가환 유니터리 변환이 된다면, 그 자체로 잡음에 면역인 양자 게이트가 된다. 다음 문서에서 마요라나 영모드와 비아벨 애니온, 위상 양자컴퓨팅을 유도한다.

---

🧩 **Principle** — 위상 불변량은 정수라서 연속 변형으로 흔들 수 없고(갭이 닫혀야만 변함), 그 이산성이 무질서 속 천 수 보존·후방산란 금지·비국소 양자 정보 저장으로 창발한다.
🔬 **Boundary** — 보호는 갭 유지·보호 대칭 유지·약한 국소 교란이라는 조건부이며, 갭이 닫히거나 대칭이 깨지거나 다체 비탄성 과정이 끼면 무너진다.
🪜 **Emergence** — 미시 잡음에서 거시 안정성이 창발하는 이 구조는 위 레이어에서 홀로그래피적 오류정정·창발 시공간 부호로 재등장하고, 저항 표준 보편성·표면 부호 오류율 $e^{-\alpha d}$ 로 검증된다.

## 📝 연습문제

**1. (기초)** "정수값을 갖는 연속 함수는 상수다" 라는 명제가 왜 위상 보호의 핵심인지 설명하고, 천 수가 변할 수 있는 유일한 통로가 무엇인지 말하라.

<details><summary>해설</summary>

천 수 $C(\lambda)$ 는 해밀토니안 변형 $\lambda$ 의 연속 함수이면서 동시에 항상 정수다. 연속 함수가 정수값만 가지면 점프할 수 없으므로(중간값 정리상 $1$ 과 $2$ 사이 값을 못 가짐) 상수여야 한다. 따라서 변형·무질서가 아무리 있어도 $C$ 는 변하지 않는다. $C$ 가 변할 수 있는 유일한 통로는 $C(\lambda)$ 의 *연속성이 깨지는 곳* — 즉 베리 곡률 적분이 발산하는 **밴드 갭이 닫히는 점**(위상 상전이). 그래서 "갭이 닫히지 않는 한 보호" 가 정확한 진술이다.
</details>

**2. (심화)** 헬리컬 가장자리에서 시간역전 불변 불순물에 의한 단일 후방산란 진폭 $\langle\downarrow|\hat V|\uparrow\rangle$ 이 0 임을 크라머스 짝 $|\downarrow\rangle=\Theta|\uparrow\rangle$ 과 반유니터리성을 써서 보여라. 짝수 번 산란은 왜 허용되는가?

<details><summary>해설</summary>

$\hat V$ 가 시간역전 불변($\Theta\hat V\Theta^{-1}=\hat V$)이고 $\Theta$ 가 반유니터리($\langle\Theta a|\Theta b\rangle=\langle b|a\rangle$)일 때: $\langle\downarrow|\hat V|\uparrow\rangle=\langle\Theta\uparrow|\hat V|\uparrow\rangle=\langle\Theta\uparrow|\Theta\hat V\Theta^{-1}|\uparrow\rangle$. 반유니터리로 $=\langle\Theta(\hat V\Theta^{-1}\uparrow)|\Theta\uparrow\rangle^*\cdots$ 를 정리하면 $=-\langle\downarrow|\hat V|\uparrow\rangle$ ($\Theta^2=-1$ 의 부호). 따라서 진폭 $=-$자기 자신 $\Rightarrow 0$. 

짝수 번 산란(예: ↑→↓→↑ 가 아니라 ↑가 불순물을 두 번 거쳐 다시 ↑)은 시간역전 짝이 아니라 *같은* 크라머스 멤버로 돌아오므로 이 상쇄가 적용되지 않아 허용된다. 다만 두 시간역전 켤레 경로의 베리 위상 $\pi$ 차이로 약반국소화(투과 강화)가 생긴다. 결국 단일 후방산란만 엄밀히 금지된다.
</details>

**3. (논문 비평)** 위상 양자 메모리(토릭 코드)는 논리 오류율이 $e^{-\alpha L}$ 로 억제된다고 한다. 그러나 실제 소자에서는 유한 온도·측정 오류·상관 잡음이 있다. 이 "지수적 보호" 주장이 실험에서 어디까지 유효하며, 어떤 가정이 깨지면 무너지는가?

<details><summary>해설</summary>

지수적 억제는 (1) 국소·독립적 오류, (2) $T=0$ (또는 갭 $\gg k_BT$), (3) 완벽한 안정자 측정, (4) 물리 오류율이 *임계값 이하* 라는 가정에서 성립한다. 깨지는 경우: (a) 유한 온도 → 열적 애니온 쌍이 생겨 갭으로 억제되지만 2D 토릭 코드는 자기정정이 안 되어 시간이 지나면 논리 오류 누적(유한 메모리 시간); (b) 상관 잡음·누설 → 오류가 비국소·상관되면 부호 거리가 보장한 보호가 약화; (c) 측정 오류 → 신드롬 추출 자체가 틀려 정정 실패율 상승, 그래서 반복 측정·실시간 디코딩 필요; (d) 물리 오류율이 임계값(표면 부호 약 1%) 초과 → $L$ 을 키워도 오류가 *증가*. 따라서 "지수적 보호" 는 임계값 이하·약상관·저온 영역에서만 유효한 *조건부* 주장이며, 실제 소자(Google 2024)가 손익분기($d$ 증가로 오류 감소)를 넘긴 것은 이 조건을 충족했음을 보인 것이다. 위상 보호의 견고함은 실재하되 무조건이 아니다.
</details>
