# 분수 양자홀 — 분수 전하·애니온

> *전자가 $e/3$ 짜리 조각으로 쪼개지고, 입자도 반입자도 아닌 통계로 서로를 감는다 — 강한 자기장 속 상호작용이 만든 창발 입자의 가장 극단적인 형태.*

## 🎯 핵심 질문

- 충전계수 $\nu=1/3$ 처럼 *부분적으로만 채워진* 란다우 준위에서, 단일입자 갭이 없는데도 *왜* 또 하나의 양자홀 고원이 생기는가?
- 라플린 바닥상태의 준입자가 *어떻게* 전자의 $1/3$ 짜리 분수 전하 $e^*=e/3$ 를 갖는가?
- 그 준입자들은 *왜* 페르미온도 보손도 아닌 **애니온(분수 통계)** 인가 — 다수의 전자에서 어떻게 전혀 새로운 통계가 창발하는가?

## 🌍 어디서 나타나나

- **초고이동도 GaAs/AlGaAs 2DEG**: Tsui, Stormer, Gossard(1982)가 $\nu=1/3$ 고원 발견(노벨상 1998). 이후 $1/5, 2/5, 3/7, 5/2,\dots$ 수십 개의 분수 고원.
- **분수 전하 직접 측정**: 양자점 접촉의 **샷 잡음(shot noise)** 이 $e^*=e/3$ 를 직접 보여줌(Saminadayar, de-Picciotto 외, 1997).
- **그래핀**: 강한 상호작용 + 약한 무질서로 $\nu=1/3$ 등 분수 양자홀 관측. $\nu=5/2$ 비아벨 후보 상태 연구의 무대.
- **냉원자·광자계**: 분수 천 절연체(FCI)를 인공 게이지장으로 합성하려는 시도 — 애니온을 통제된 환경에서 구현하려는 최전선.

## 🔍 직관의 함정

**함정 1 — "전하 $e/3$ 은 전자가 진짜 쪼개진 것이다."** 아니다. 어떤 단일 전자도 쪼개지지 않는다. $e/3$ 은 **다체 바닥상태의 집단적 들뜸(준입자)** 이 나르는 전하다. 전자 $N$ 개의 얽힌 액체에서, 국소 결손(quasihole)이 정확히 전자의 $1/3$ 만큼 전하를 비운다. 창발적 실체이지 근본 입자의 분할이 아니다.

**함정 2 — "통계는 페르미온 아니면 보손뿐."** 그건 3차원의 정리다. **2차원에서는 입자 교환이 위상적으로 풍부**해서(두 입자 교환 경로가 홀로토피적으로 비자명), 교환 위상이 $0$(보손)이나 $\pi$(페르미온)가 아닌 임의의 값 $\theta$ 일 수 있다 — 애니온. 분수 양자홀이 그 첫 실현.

**함정 3 — "분수 양자홀은 정수 양자홀의 작은 보정이다."** 정반대로 *질적으로 다른* 상이다. 정수 양자홀은 단일입자(채워진 밴드)로 설명되지만, 분수 양자홀은 **전자–전자 상호작용 없이는 존재할 수 없다.** 갭은 상호작용이 만들고, 위상 질서(긴 거리 얽힘)가 본질이다.

## ⚙️ 제1원리 유도

### 1) 최저 란다우 준위와 대칭 게이지 — 다항식 × 가우시안

대칭 게이지 $\mathbf{A}=\frac{B}{2}(-y,x)$ 에서 최저 란다우 준위(LLL)의 단일입자 상태는 복소좌표 $z=x-iy$ 로

$$
\psi_m(z)\propto z^m\,e^{-|z|^2/4\ell_B^2},\qquad \ell_B=\sqrt{\hbar/eB},
$$

자기 길이 $\ell_B$. LLL 안의 임의 상태는 $z$ 의 **해석적 다항식 × 가우시안** 이다. 이것이 라플린의 출발점이다(이하 $\ell_B=1$).

### 2) 라플린 파동함수 — 추측이 아니라 제약의 결과

$\nu=1/m$ ($m$ 홀수) 의 다체 바닥상태를 짓자. 요구 조건:
(i) LLL 안 → 모든 $z_i$ 의 해석적 다항식 × $e^{-\sum|z_i|^2/4}$,
(ii) 페르미온 반대칭 → $z_i\leftrightarrow z_j$ 교환 시 부호 반전,
(iii) 전 각운동량 고정 → 동차 다항식,
(iv) 입자가 가까이 오는 것을 최대한 피함(상호작용 에너지 최소화) → 두 입자가 겹칠 때 빠르게 0.

이 제약을 모두 만족하는 가장 간단한 형태:

$$
\boxed{\Psi_m(z_1,\dots,z_N)=\prod_{i<j}(z_i-z_j)^m\,\exp\!\left(-\frac{1}{4}\sum_i|z_i|^2\right).}\quad\square
$$

$m$ 이 홀수면 교환 시 $(z_i-z_j)^m\to(-1)^m(\cdots)=-(\cdots)$ 로 반대칭(페르미온). 지수 $m$ 이 클수록 입자들이 더 멀리 떨어져 쿨롱 에너지를 낮춘다. $\nu=1/m$ 임은 다음 단계에서 나온다.

### 3) 플라스마 유추 — 충전계수가 $1/m$ 인 이유

$|\Psi_m|^2$ 의 로그를 보자:

$$
|\Psi_m|^2=\exp\!\left[2m\sum_{i<j}\ln|z_i-z_j|-\frac{1}{2}\sum_i|z_i|^2\right]\equiv e^{-\beta U}.
$$

이것은 2차원 **단일성분 플라스마**(전하 $m$ 의 입자들 + 균일 배경)의 볼츠만 가중치와 같다. 플라스마가 전하 중성이 되려면 입자 밀도가

$$
n=\frac{1}{2\pi m\ell_B^2}=\frac{1}{m}\cdot\frac{eB}{h}=\frac{1}{m}\,n_\Phi,
$$

즉 충전계수 $\nu=n/n_\Phi=1/m$.

$$
\boxed{\nu=\frac{1}{m}\quad(\text{$\nu=1/3$ 은 $m=3$}).}\quad\square
$$

### 4) 준구멍과 분수 전하 — $e^* = e/m$

점 $\xi$ 에 자기 선속 한 양자를 단열적으로 끼워넣으면(또는 동치로 곱셈 연산자 적용) 준구멍(quasihole) 상태가 생긴다:

$$
\Psi_{\text{qh}}=\prod_i(z_i-\xi)\,\Psi_m.
$$

이 인자는 $\xi$ 근처에서 전자 밀도를 비운다. 플라스마 유추로 비워진 전하를 계산하자: 선속 양자 하나를 끼우면 정수 양자홀에서는 전하 $e$ 가 옮겨지지만, $\nu=1/m$ 에서는 **전하 $e/m$** 만 비워진다. 이유: 단위 선속당 밀도 변화가 $\nu=1/m$ 배이기 때문.

$$
\boxed{e^*=\nu\,e=\frac{e}{m}\quad(\nu=1/3\;\Rightarrow\;e^*=e/3).}\quad\square
$$

세 개의 준구멍을 합치면 정확히 전자 하나의 빈자리($e$)가 된다 — $3\times(e/3)=e$. 분수 전하는 정수 전하의 모순이 아니라 *집단적 들뜸* 의 성질이다.

### 5) 분수 통계 — 두 준구멍을 감으면 위상 $\theta=\pi/m$

두 준구멍을 $\xi_1,\xi_2$ 에 두고, 하나를 다른 하나 둘레로 한 바퀴 돌리자. 베리 위상은 두 부분으로 나뉜다:
(a) 끼운 선속에 의한 아하로노프–봄 위상(둘러싼 전하에 비례),
(b) 두 준구멍 사이의 **상대 통계 위상**.

플라스마 유추로 베리 위상을 계산하면, $\xi_2$ 가 $\xi_1$ 을 한 바퀴 감을 때 추가 위상

$$
\Delta\gamma_{\text{braid}}=2\pi\cdot\frac{1}{m}\quad\Longrightarrow\quad
\theta_{\text{exchange}}=\frac{\pi}{m}.
$$

한 번 *교환*(반바퀴)은 위상 $\theta=\pi/m$. $m=3$ 이면 $\theta=\pi/3$ — 보손($0$)도 페르미온($\pi$)도 아닌 **애니온**.

$$
\boxed{\theta=\frac{\pi}{m}\quad(\nu=1/3\;\Rightarrow\;\theta=\pi/3,\ \text{아벨 애니온}).}\quad\square
$$

이것은 *아벨* 애니온(위상이 단순 곱셈). 비아벨 통계(브레이딩이 행렬)는 $\nu=5/2$ 같은 후보 상태에서 나오며 다음 챕터 [06](./06-majorana-anyon.md) 의 주제다.

### 실험 1 — 라플린 상태의 충전계수와 분수 전하 시연

```python
import numpy as np

# 라플린 nu=1/m 상태: 플라스마 유추로 충전계수와 준구멍 전하 확인.
# 작은 N 입자계에서 |Psi|^2 의 밀도 프로파일을 몬테카를로로 추정.

def laughlin_logprob(z, m):
    """log|Psi_m|^2 = 2m Σ ln|zi-zj| - (1/2)Σ|zi|^2  (ℓ_B=1)"""
    N = len(z)
    s = 0.0
    for i in range(N):
        for j in range(i+1, N):
            s += 2*m*np.log(abs(z[i]-z[j]) + 1e-12)
    s -= 0.5*np.sum(np.abs(z)**2)
    return s

def metropolis(m, N=10, steps=40000, seed=0):
    rng = np.random.default_rng(seed)
    z = (rng.standard_normal(N) + 1j*rng.standard_normal(N)) * np.sqrt(m*N)
    logp = laughlin_logprob(z, m)
    samples = []
    for t in range(steps):
        i = rng.integers(N)
        zt = z.copy()
        zt[i] += (rng.standard_normal() + 1j*rng.standard_normal()) * 1.0
        logpt = laughlin_logprob(zt, m)
        if np.log(rng.random()) < logpt - logp:
            z, logp = zt, logpt
        if t % 50 == 0:
            samples.append(z.copy())
    return np.array(samples)

for m in [1, 3]:
    s = metropolis(m, N=10, steps=20000)
    r = np.abs(s).flatten()
    # nu=1/m 이면 밀도 ~ 1/m → 같은 입자수가 더 넓게 퍼진다(반지름 ∝ √m).
    print(f"m={m} (nu=1/{m}): 입자 분포 반지름 중앙값≈{np.median(r):.1f}, 최대≈{r.max():.1f}")
# m=3 이 m=1 보다 분포가 √3 배가량 더 퍼짐 → 밀도 1/3 → 충전계수 nu=1/3 확인
# (유한 N·경계 효과로 절대값은 열역학 극한 예측 √(2mN) 과 다소 차이)
```

### 실험 2 — 2차원 교환 통계: 보손/페르미온/애니온 위상

```python
import numpy as np

# 두 애니온을 반바퀴(교환) 돌릴 때 누적되는 베리 위상을 모형화.
# 아벨 애니온: 파동함수가 e^{i*theta} 만큼 위상 획득.
def exchange_phase(m):
    """nu=1/m 라플린 준구멍의 교환 통계각 theta = pi/m"""
    return np.pi / m

for label, m in [("보손", None), ("페르미온", 1), ("애니온 nu=1/3", 3), ("애니온 nu=1/5", 5)]:
    if m is None:
        theta = 0.0
    else:
        theta = exchange_phase(m)
    print(f"{label:16s}: 교환위상 theta = {theta/np.pi:.3f} π,  e^{{iθ}} = {np.exp(1j*theta):+.3f}")
# 보손 0, 페르미온 π(부호 -1), nu=1/3 은 π/3 — 그 중간의 분수 통계
```

## 🧪 실험·관측 증거

- **분수 전하 샷 잡음**: $\nu=1/3$ 에서 양자점 접촉을 통한 터널링 전류의 잡음이 $S=2e^*I$ ($e^*=e/3$)를 줌(de-Picciotto 외, *Nature* 1997; Saminadayar 외 1997). 분수 전하의 직접 측정.
- **간섭계 분수 통계**: Fabry–Pérot/Mach–Zehnder 애니온 간섭계에서 $\nu=1/3$ 의 통계각 $\theta=\pi/3$ 관측(Nakamura 외, *Nature Physics* 2020) — 애니온 통계의 직접 증거.
- **분수 고원**: $\nu=1/3,2/5,3/7,\dots$ 의 정확한 양자화 $R_{xy}=h/(\nu e^2)$ — 조던–HLR 합성 페르미온 계열로 설명.
- **에너지 갭**: $\nu=1/3$ 갭 $\sim 0.1\,e^2/\epsilon\ell_B$ — 상호작용 스케일. 단일입자 갭이 아님을 확인.

## 🔬 경계

- **라플린 상태는 모형 상태**: 실제 쿨롱 상호작용의 정확한 바닥상태가 아니라 *그것과 같은 위상 상(universality class)* 에 있는 변분 상태. 분수 양자홀의 위상은 견고하지만 라플린 파동함수 자체는 근사.
- **짝수 분모($\nu=5/2$)**: 라플린 형태(홀수 $m$)로 안 됨. Moore–Read Pfaffian 등 비아벨 후보가 필요 — 미해결 부분이 남아 있음(→ [06](./06-majorana-anyon.md)).
- **무질서·온도**: 분수 갭이 작아($\sim$ K 이하) 초저온·초고이동도가 필수. 무질서가 강하면 위상 상이 무너진다.
- **3차원에는 없음**: 애니온은 2차원 현상. 3차원에서는 교환 경로가 풀려 페르미온/보손만 남는다.

## 🧬 횡단 원리

- **(→ quantum-information)**: 애니온의 분수/비아벨 통계는 위상 양자컴퓨팅의 핵심 자원. 분수 전하·통계는 *국소적으로 측정·교란할 수 없는* 정보 — 위상 보호(→ [05](./05-topological-protection.md))의 가장 극적 형태.
- **(→ ch5 분수화)**: 전자가 분수 전하 준입자로 "쪼개지는" 것은 분수화(fractionalization)의 원형 — 스핀–전하 분리, 창발 게이지장과 같은 가족(→ [../ch5-emergent-gauge/02-fractionalization.md](../ch5-emergent-gauge/02-fractionalization.md)).
- **(→ entanglement)**: 분수 양자홀의 본질은 긴 거리 얽힘(위상 질서). 바닥상태 겹침·위상 엔트로피로 정량화(→ [../ch5-emergent-gauge/04-entanglement-phases.md](../ch5-emergent-gauge/04-entanglement-phases.md)).

## 🪜 창발

- **(L5 holography)**: 분수 양자홀의 가장자리는 카이럴 Luttinger 액체(분수 전하 들뜸의 CFT). 벌크 위상 질서 ↔ 경계 CFT 의 대응은 홀로그래피적 벌크–경계 사전의 응집물질 예시.
- **(L6 emergence)**: "전하는 $e$ 의 정수배" 라는 근본 규칙이 다수의 얽힘에서 *분수* 로 창발하는 것 — 근본 양자수조차 집단 현상 앞에서 재정의됨. "More is Different" 의 가장 강한 형태.

## 📐 예측·반증

- **정량 예측**: $\nu=1/3$ 준입자 전하 $e^*=e/3$, 교환 통계각 $\theta=\pi/3$. 세 준입자가 모이면 정확히 전자 빈자리 하나($e$).
- **반증 가능성**: 만약 분수 전하가 측정 오차의 산물이라면, 샷 잡음 기울기 $S/I=2e^*$ 가 $\nu$ 에 따라 정확히 $2(e/m)$ 로 변하지 않아야 한다. 실측은 정확히 변함 — 분수 전하를 지지.
- **간섭계 검증**: 애니온이 다른 애니온 둘레를 한 바퀴 돌면 간섭 위상이 $2\pi/m$ 만큼 이산적으로 점프해야 한다. 관측 일치.

분수 양자홀은 시간역전 대칭을 *깨는*(자기장) 위상 상이었다. 그런데 시간역전 대칭을 *보존하면서도* 위상적으로 비자명한 상이 있을까? 천 수가 0으로 강제되는 그곳에서 새 불변량 $\mathbb{Z}_2$ 가 등장하고, "안은 절연체·겉은 도체" 인 위상 절연체가 나온다. 다음 문서의 주제다.

🧩 **Principle** — 강한 자기장 속 상호작용 전자의 얽힌 액체(라플린 상태)에서 분수 전하 $e/m$ 준입자와 분수 통계 애니온이 창발하며, 충전계수·전하·통계가 모두 단일 정수 $m$ 으로 묶인다.
🔬 **Boundary** — 라플린 형태는 홀수 $m$ 의 모형 상태로 짝수 분모($\nu=5/2$)에는 비아벨 기술이 필요하고, 작은 갭 탓에 초저온·고이동도·약무질서가 필수이며 애니온은 2차원에만 존재한다.
🪜 **Emergence** — 근본 전하 $e$ 가 다수의 얽힘에서 분수로 재정의되는 이 구조는 위 레이어에서 분수화·위상 양자컴퓨팅·홀로그래피적 경계 CFT 로 재등장하고, $e^*=e/3$ 샷 잡음·$\theta=\pi/3$ 간섭으로 검증된다.

## 📝 연습문제

**1. (기초)** 라플린 $\nu=1/m$ 파동함수에서 $m$ 이 *홀수* 여야 하는 이유를 전자의 페르미온 통계로 설명하라. $m=2$ 이면 어떤 입자의 바닥상태가 되는가?

<details><summary>해설</summary>

전자는 페르미온이라 두 입자 교환 시 부호가 반전되어야 한다. $\prod(z_i-z_j)^m$ 에서 $z_i\leftrightarrow z_j$ 교환은 $(z_i-z_j)^m\to(-1)^m(z_i-z_j)^m$. 반대칭이려면 $(-1)^m=-1$, 즉 $m$ 홀수. $m=2$ (짝수)면 교환 시 부호 불변 → *보손* 의 바닥상태(예: 회전하는 보스 기체의 분수 양자홀 유사 상태). 그래서 전자 분수 양자홀의 라플린 계열은 $m=1,3,5,\dots$ 만 허용된다.
</details>

**2. (심화)** 준구멍 $\Psi_{\text{qh}}=\prod_i(z_i-\xi)\Psi_m$ 이 $\xi$ 근처에서 비우는 전하가 $e/m$ 임을, 플라스마 유추(전하 중성 조건)를 써서 정량적으로 논증하라.

<details><summary>해설</summary>

$|\Psi_{\text{qh}}|^2=e^{-\beta U'}$ 에서 추가 항 $\sum_i 2\ln|z_i-\xi|=\frac{2}{m}\cdot m\sum_i\ln|z_i-\xi|$ 는 플라스마에서 $\xi$ 에 놓인 *전하 $1$* 짜리 불순물(플라스마 입자는 전하 $m$)에 해당한다. 플라스마는 전하 중성을 회복하려 이 불순물을 차폐하는데, 차폐 구름의 전하는 $-1$ (플라스마 단위). 플라스마 입자 하나가 전자 하나에 대응하고 전자 전하가 $e$ 이므로, 비워진 물리적 전하는 $\frac{1}{m}\times e=e/m$. 곧 준구멍은 전자의 $1/m$ 만큼 전하를 갖는다. ($m=3\Rightarrow e/3$.)
</details>

**3. (논문 비평)** Nakamura 외(2020)는 애니온 간섭계로 $\nu=1/3$ 의 통계각 $\theta=\pi/3$ 를 보고했다. 그러나 이런 간섭 실험은 쿨롱 충전 효과·에지 간섭과 분수 통계를 분리하기 어렵다는 비판이 있다. 분수 통계를 *명확히* 입증하려면 어떤 대조 실험이 필요한가?

<details><summary>해설</summary>

핵심 난점은 (a) 간섭 위상 점프가 진짜 애니온 *브레이딩* 때문인지 (b) 갇힌 영역의 쿨롱 충전(전하 한 개 추가 시 위상 점프) 때문인지 구별이다. 대조 전략: (1) 간섭 고리 안 애니온 수 $N_{\text{qp}}$ 를 게이트로 *하나씩* 바꾸며 위상 점프가 정확히 $2\pi/m$ 단위(브레이딩 예측)로 양자화되는지 확인; (2) 충전 효과는 면적·정전용량에 의존하므로 소자 크기를 바꿔 그 의존성을 분리; (3) 정수 양자홀($\nu=1$, 페르미온, 점프 $2\pi$)과 분수($\nu=1/3$, 점프 $2\pi/3$)를 같은 소자에서 비교. 위상 점프가 *위상 질서가 예측하는 분수값* 이고 충전 효과로 설명 안 되면 분수 통계가 입증된다. 비아벨($\nu=5/2$)에서는 더 나아가 브레이딩 순서가 결과를 바꾸는 비가환성까지 보여야 한다(→ [06](./06-majorana-anyon.md)).
</details>
