# 창발 게이지장 — 근본 vs 창발의 역설

> *스핀 액체에서 광자와 U(1)·Z₂ 게이지장이 *맨몸의 스핀에서* 창발한다 — 게이지 대칭조차 근본이 아니라 다체 얽힘의 산물일 수 있다.*

## 🎯 핵심 질문

- 격자 위의 스핀(전자도, 전하도, 광자도 없는 순수 스핀)만 있는 모형에서 어떻게 **광자**와 **게이지장**이 *나오는가*? 무엇이 "$\nabla\cdot E=0$" 같은 가우스 법칙을 강제하는가?
- 전자를 스피논·홀론으로 분해(slave-particle)할 때 나타나는 **게이지 자유도**는 단순한 계산 기교인가, 아니면 측정 가능한 물리인가?
- 표준모형의 게이지장은 "근본"으로 가정된다. 그런데 응집물질에서 게이지장이 *창발* 한다면 — **무엇이 진짜 근본인가**? 우리 우주의 전자기장도 창발일 수 있는가?

## 🌍 어디서 나타나나

창발 게이지장은 추상이 아니라 실험실의 물질에서 작동한다.

- **양자 스핀 액체(QSL)**: $\alpha$-RuCl₃, 헤르베르츠마이트(herbertsmithite, ZnCu₃(OH)₆Cl₂), 유기 모트 절연체 $\kappa$-(ET)₂Cu₂(CN)₃. 반강자성 교환에도 불구하고 절대영도까지 자기 질서가 *얼지 않는* 모트 절연체. 그 바닥상태는 긴 거리 얽힘을 가지며, 들뜸은 창발 게이지장에 결합한 분수 입자다.
- **스핀 아이스(spin ice)**: Dy₂Ti₂O₇, Ho₂Ti₂O₇. 파이로클로어 격자에서 "2-in-2-out" 얼음 규칙이 창발 U(1) 게이지장의 가우스 법칙을 그대로 구현하고, 들뜸은 **창발 자기 단극**으로 행동한다 (Castelnovo–Moessner–Sondhi, 2008).
- **양자 자기 단극**: 스핀 아이스에서 단극의 운동이 만드는 "자기류(magnetricity)" 가 측정되었다 (Bramwell et al., 2009).
- **양자 시뮬레이터**: 초전도 큐비트·중성원자 격자에서 Z₂ 게이지 이론(토릭 코드 계열)을 직접 구현해 게이지 구조를 합성한다.

## 🔍 직관의 함정

- **"게이지장은 손으로 넣어야 나온다"** — 아니다. 스핀 모형의 해밀토니안에는 게이지장이 *없다*. 게이지장은 다체 바닥상태의 저에너지 들뜸으로 **나온다**. 손으로 넣은 것이 아니라 창발한 것이다.
- **"슬레이브 입자의 게이지장은 그냥 표기 중복(redundancy)"** — 부분적으로만 맞다. 분해 $c=b^\dagger f$ 는 국소 위상 자유도라는 *수학적* 중복을 도입하지만, 그 중복을 다루는 게이지장이 *동역학적* 으로 되면(요동이 살아남으면) 물리적 게이지 보손이 된다. 닫힘(confine)/풀림(deconfine)이 상을 가른다.
- **"창발 광자는 비유다"** — 아니다. 창발 광자는 선형 분산 $\omega=c_{\text{eff}}\,k$, 두 횡편광, 가우스 법칙까지 진짜 광자의 구조를 갖는다. 다만 빛의 속도 $c$ 대신 유효 속도 $c_{\text{eff}}$(보통 격자 에너지/거리 척도)로 달린다.
- **"근본 = 더 진짜"** — 이 문서가 깨려는 핵심 오해. 창발 게이지장이 근본 게이지장과 구조적으로 구별 불가능하다면, "근본" 은 형이상학적 라벨일 뿐 물리적 차이가 아닐 수 있다.

## ⚙️ 제1원리 유도

### 1. 슬레이브-입자 분해와 게이지 중복의 탄생

모트 절연체(반쪽 채움, 한 사이트에 전자 1개)에서 전자 연산자 $c_{i\sigma}$ 를 두 조각으로 쪼갠다:

$$c_{i\sigma} = b_i^\dagger \, f_{i\sigma}$$

여기서 $f_{i\sigma}$ 는 전하를 잃은 중성 스핀-1/2 페르미온(**스피논**), $b_i$ 는 스핀을 잃은 전하 $+e$ 보손(**홀론**)이다. 이 분해는 다음 **국소 제약** 하에서만 원래 힐베르트 공간과 일치한다 (사이트당 입자 하나):

$$b_i^\dagger b_i + \sum_\sigma f_{i\sigma}^\dagger f_{i\sigma} = 1$$

여기서 결정적 관찰: 임의의 사이트별 위상 $\theta_i(\tau)$ 에 대해

$$f_{i\sigma}\to e^{i\theta_i}f_{i\sigma}, \qquad b_i \to e^{i\theta_i} b_i$$

는 물리적 전자 $c_{i\sigma}=b_i^\dagger f_{i\sigma}$ 를 **불변**으로 남긴다 ($e^{-i\theta_i}e^{i\theta_i}=1$). 즉 분해는 국소 $U(1)$ **게이지 대칭**을 *자동으로* 가진다. 이 대칭은 물리에 없던 것을 우리가 도입한 중복이지만 — 바로 그 중복을 추적하는 보상장이 게이지장 $a_{ij}$ 다. $\square$

### 2. 교환 해밀토니안에서 게이지장이 솟아오른다

하이젠베르크 모형 $H=J\sum_{\langle ij\rangle}\mathbf{S}_i\cdot\mathbf{S}_j$ 에서 스핀을 스피논으로 표현한다:

$$\mathbf{S}_i = \tfrac{1}{2}\sum_{\alpha\beta} f_{i\alpha}^\dagger \,\boldsymbol{\tau}_{\alpha\beta}\, f_{i\beta}$$

($\boldsymbol{\tau}$ 는 파울리 행렬). 교환항을 페르미온 쌍선형으로 펼치면 4-페르미온 상호작용이 나오고, 이를 평균장으로 분해한다. 결합장(bond field)

$$\chi_{ij} = \langle \textstyle\sum_\sigma f_{i\sigma}^\dagger f_{j\sigma}\rangle = |\chi_{ij}|\,e^{i a_{ij}}$$

를 도입하면, 스피논은 *호핑 진폭* $|\chi_{ij}|$ 와 *위상* $a_{ij}$ 위를 움직인다. 평균장 해밀토니안은

$$H_{\text{MF}} \sim -t\sum_{\langle ij\rangle}\big(f_{i\sigma}^\dagger\,e^{i a_{ij}}\,f_{j\sigma} + \text{h.c.}\big)$$

이때 $a_{ij}$ 는 격자 위의 **벡터 퍼텐셜** 처럼 들어간다. 게이지 변환 $f_i\to e^{i\theta_i}f_i$ 는 정확히

$$a_{ij}\to a_{ij} + \theta_i - \theta_j$$

로 보상되어야 $H_{\text{MF}}$ 가 불변이다 — 이는 격자 $U(1)$ 게이지장의 정의 그 자체다. 처음에 순수 스핀만 있던 모형에서 격자 게이지 퍼텐셜 $a_{ij}$ 가 *동역학적 자유도로 출현* 했다. $\square$

### 3. 가우스 법칙과 창발 전기장

게이지장 $a_{ij}$ 와 짝을 이루는 정준 켤레가 격자 **전기장** $E_{ij}$ 다. 사이트별 제약(입자 수 보존)은 라그랑주 승수 $a_0(i)$ 를 통해 들어오고, 이는 정확히 **가우스 법칙**

$$\sum_{j\in i}E_{ij} = \rho_i \;=\; q_i - \bar q$$

를 강제한다. 여기서 $\rho_i$ 는 사이트의 스피논 전하다. 즉 "사이트당 1입자" 라는 미시 제약이 저에너지에서 게이지 이론의 가우스 법칙으로 *번역* 된다. 가우스 법칙은 손으로 부과한 것이 아니라 미시 제약의 창발적 결과다. $\square$

### 4. 두 상: 닫힘(confined) vs 풀림(deconfined)

격자 $U(1)$ 게이지 이론(Polyakov, Fradkin–Shenker)은 두 상을 가진다. 컴팩트 $U(1)$ 게이지장의 유효 작용을 도입하면, 게이지장 요동의 강성 $\kappa$ 가 임계값을 넘느냐에 따라:

- **닫힘 상**: 모노폴(instanton)이 응축, 스피논 사이에 선형 퍼텐셜 $V(r)\sim\sigma r$ 이 생겨 스피논이 *쌍으로 묶인다*. 저에너지에 게이지 보손 없음 → 보통의 자기 질서/VBS.
- **풀림 상**: 모노폴이 억제, 스피논이 자유로워지고 **무질량 광자**(컴팩트 $U(1)$ 의 횡파 게이지 모드)가 살아남는다. 이것이 **창발 광자** 를 가진 $U(1)$ 스핀 액체다.

3+1차원 컴팩트 $U(1)$ 에서는 풀림 상이 안정해 진짜 무질량 광자가 존재한다 (스핀 아이스가 이 부류). $\square$

### 5. Z₂ 게이지장: 더 견고한 창발

$U(1)$ 대신 **Z₂** 게이지 구조도 창발한다. 스피논 쌍이 응축하면($\langle b b\rangle\neq0$) 연속 $U(1)$ 이 이산 $Z_2$ 로 깨진다(앤더슨-힉스). Z₂ 게이지장은 모노폴 불안정이 없어 2+1차원에서도 **풀림 상이 안정**하다. 이때 들뜸은:

- $Z_2$ 전하: 스피논(spinon) — 게이지 전기 입자
- $Z_2$ 자기선속: **비손(vison)** — 게이지 자기 입자

스피논과 비손은 서로를 감으면 $-1$ 의 위상을 얻는(상호 준통계) 애니온이다. 이 $Z_2$ 스핀 액체가 **토릭 코드** 와 같은 보편 부류이며, 다음 문서들(분수화·위상 질서)의 무대가 된다. $\square$

### 실험 1 — 격자 U(1) 게이지장의 가우스 법칙 검증

순수 스핀 제약(사이트당 입자 수)이 가우스 법칙으로 번역됨을, 작은 격자에서 게이지장 위상에 무관한 물리(게이지 불변량)로 확인한다.

```python
import numpy as np

# 1D 고리 위 스피논 호핑 + U(1) 게이지장 a_ij (링크 위상)
# H_MF = -t sum_j (f_j^† e^{i a_j} f_{j+1} + h.c.)
# 게이지 변환 f_j -> e^{i θ_j} f_j,  a_j -> a_j + θ_j - θ_{j+1}
# 물리량(에너지 스펙트럼)은 게이지 변환에 불변이어야 한다.

def build_H(N, t, a):
    H = np.zeros((N, N), dtype=complex)
    for j in range(N):
        jp = (j + 1) % N
        H[j, jp] += -t * np.exp(1j * a[j])
        H[jp, j] += -t * np.exp(-1j * a[j])
    return H

N, t = 8, 1.0
rng = np.random.default_rng(0)

# 임의의 게이지장 a
a = rng.uniform(-np.pi, np.pi, N)
E0 = np.linalg.eigvalsh(build_H(N, t, a))

# 임의의 게이지 변환 θ 적용
theta = rng.uniform(-np.pi, np.pi, N)
a_gauge = np.array([a[j] + theta[j] - theta[(j+1) % N] for j in range(N)])
E1 = np.linalg.eigvalsh(build_H(N, t, a_gauge))

print("스펙트럼 차이(게이지 불변 확인):", np.max(np.abs(E0 - E1)))
# 게이지 변환 후에도 물리량(에너지)은 불변 -> ~1e-14

# 물리적 게이지 불변량 = 고리를 감는 총 선속(Wilson loop)
flux = np.sum(a) % (2*np.pi)
flux_gauge = np.sum(a_gauge) % (2*np.pi)
print("Wilson loop 선속(불변):", flux, flux_gauge)
# θ 의 합은 고리를 따라 상쇄 -> 선속 동일
```

게이지 변환은 스펙트럼을 바꾸지 않지만(중복), 고리를 감는 **Wilson 선속** 은 게이지 불변 물리량으로 살아남는다 — 창발 게이지장의 실재를 보여준다.

### 실험 2 — Fradkin–Shenker 위상도: 닫힘 vs 풀림

2+1차원 Z₂ 게이지-힉스 모형의 위상 구조를 작은 격자에서 Wilson 선속 요동으로 거칠게 진단한다.

```python
import numpy as np

# Z₂ 게이지장: 각 링크의 부호 s_ij ∈ {+1,-1}
# 작용 S = -K Σ_plaq (∏ s) - h Σ_link s  (단순화)
# 몬테카를로로 평균 플라켓(닫힘/풀림 진단) 측정
def plaquette(s):
    # s: shape (L,L,2)  s[...,0]=x링크, s[...,1]=y링크
    L = s.shape[0]
    P = (s[:, :, 0] *
         np.roll(s[:, :, 1], -1, axis=0) *
         np.roll(s[:, :, 0], -1, axis=1) *
         s[:, :, 1])
    return P

def run(L, K, h, steps=4000, seed=0):
    rng = np.random.default_rng(seed)
    s = rng.choice([-1, 1], size=(L, L, 2))
    def energy():
        return -K * plaquette(s).sum() - h * s.sum()
    E = energy(); acc = []
    for _ in range(steps):
        i, j, mu = rng.integers(L), rng.integers(L), rng.integers(2)
        s[i, j, mu] *= -1
        En = energy()
        if rng.random() < np.exp(-(En - E)):
            E = En
        else:
            s[i, j, mu] *= -1
        acc.append(plaquette(s).mean())
    return np.mean(acc[steps//2:])

for K, h in [(0.2, 0.1), (2.0, 0.1), (2.0, 2.0)]:
    P = run(8, K, h)
    phase = "풀림(낮은 플라켓 에너지)" if P > 0.5 else "무질서"
    print(f"K={K}, h={h}: <plaquette>={P:+.3f}  ({phase})")
# K 큰 영역 -> 플라켓 정렬(풀림), 게이지장 요동이 살아남는 상
```

## 🧪 실험·관측 증거

- **헤르베르츠마이트**: 카고메 격자 $S=1/2$ 반강자성체. 중성자 산란에서 *연속* 스펙트럼(뾰족한 마그논이 아닌 연속체)이 관측 — 분수 스피논과 게이지장 결합의 징표 (Han et al., Nature 2012).
- **스핀 아이스 단극**: Dy₂Ti₂O₇ 에서 창발 자기 단극의 운동(자기류)이 측정됨 (Bramwell et al., Nature 2009). 이는 창발 $U(1)$ 게이지장의 가우스 법칙이 물질에서 작동함을 의미한다.
- **$\alpha$-RuCl₃**: 키타에프 자성체 후보. 반정수 양자화된 열홀 전도도가 보고되어(다소 논쟁적) 마요라나 + 게이지선속(vison) 구조와 연결된다 (Kasahara et al., 2018).
- **양자 시뮬레이션**: 초전도 큐비트로 토릭 코드(Z₂ 게이지) 바닥상태를 준비하고 애니온 브레이딩을 시연 (Google, Nature 2021 계열).

## 🔬 경계

- **슬레이브-입자 평균장의 한계**: 분해는 정확하지만 평균장 근사는 제약 $b^\dagger b+\sum f^\dagger f=1$ 을 *평균적으로만* 강제한다. 게이지 요동을 무시하면 닫힘/풀림을 잘못 판정할 수 있다. 게이지장을 동역학적으로 다뤄야(요동 포함) 진짜 상이 결정된다.
- **2+1D 컴팩트 U(1) 의 불안정**: 순수 컴팩트 $U(1)$ 은 2+1차원에서 모노폴 응축으로 *항상 닫힘* (Polyakov). 풀림 $U(1)$ 스핀 액체를 2D에서 안정화하려면 충분히 많은 무질량 물질장이 모노폴을 억제해야 한다 — 미묘한 임계 문제.
- **실험적 식별의 어려움**: "자기 질서의 부재" 만으로 스핀 액체를 단정할 수 없다(단순 무질서·유리 상태와 구별 필요). 분수화·게이지장의 *양성* 증거(연속 스펙트럼, 열홀, 얽힘 엔트로피)가 필요하다.
- **고온 초전도와의 연결**: RVB·슬레이브-보손 이론이 고온 초전도의 유력 후보지만, 미해결이다(→ Ch6-04).

## 🧬 횡단 원리

- **대칭(SSB)과 대비**: 게이지 "대칭" 은 진짜 대칭이 아니라 기술의 중복이다(엘리추르 정리: 국소 게이지 대칭은 자발적으로 깨지지 않는다). 그래서 창발 게이지 상은 SSB 패러다임 *밖* 의 위상 질서다(→ ch5-emergent-gauge/03-topological-order.md).
- **정보(얽힘)**: 풀림 상의 긴 거리 얽힘이 게이지장의 무질량성과 가우스 법칙을 떠받친다. 얽힘 엔트로피의 위상항 $\gamma=\ln\mathcal{D}$ 가 게이지 군의 차원을 센다(→ ch5-emergent-gauge/04-entanglement-phases.md).
- **분수화**: 게이지장에 결합한 입자가 *분수* 전하·통계를 갖는 것이 분수화다(→ ch5-emergent-gauge/02-fractionalization.md, → ch4-topological-matter/03-fractional-quantum-hall.md).
- **표준모형과의 대비**: 표준모형 $SU(3)\times SU(2)\times U(1)$ 게이지장은 "근본" 으로 가정된다. 창발 게이지의 존재는 이 가정에 의문을 제기한다(→ standard-model, L6).

## 🪜 창발

- **L6 emergence**: 창발 게이지장은 "More is Different" 의 가장 극적인 사례다. *대칭 원리* 그 자체가 미시에 없던 것이 다체 얽힘에서 *나온다*. 게이지 원리가 근본 공리가 아니라 창발적 조직 원리일 수 있다는 가능성.
- **standard-model(L3)**: Wen 등은 우주의 모든 기본 입자(광자·전자·쿼크·글루온)가 거대한 큐비트 그물(string-net, 양자 정보)의 창발 들뜸일 수 있다는 그림을 제시한다. 그렇다면 "근본 vs 창발" 의 구분 자체가 사라진다.
- **L5 holography**: 강상관계의 게이지장(이미 창발)이 *추가 차원의 기하*로 다시 창발하는 게이지/중력 이중성으로 연결된다(→ ch5-emergent-gauge/04-entanglement-phases.md).

## 📐 예측·반증

- **창발 광자의 선형 분산**: $U(1)$ 스핀 액체는 저에너지에 무질량 광자 모드 $\omega=c_{\text{eff}}k$ 를 가진다. 비열 $C\sim T^3$(3D 광자) 기여를 예측 — 측정으로 반증 가능.
- **스핀 아이스 자기 단극**: 창발 단극은 쿨롱 상호작용 $V(r)\sim 1/r$ 을 따라야 한다. 자기류·자화 완화 측정으로 검증됨.
- **Z₂ 스핀 액체의 4겹 바닥상태(토러스)**: 위상 질서는 토러스에서 $\mathcal{D}^2=4$겹 바닥상태 겹침을 예측(→ 03). 수치(DMRG)로 확인 가능.
- **반증 조건**: 어떤 후보 스핀 액체에서 (1) 자기 질서가 결국 나타나거나, (2) 연속 스펙트럼이 단순 무질서로 설명되거나, (3) 위상 얽힘 엔트로피 $\gamma\neq\ln\mathcal{D}$ 이면, 창발 게이지 시나리오는 그 물질에서 반증된다.

## 🤔 다음 질문

게이지장이 창발한다면, 게이지장에 결합한 **입자** 는 어떻게 보이는가? 전자가 스피논·홀론으로 *쪼개지는* 분수화는 어떻게 가능하고, 1차원에서 스핀과 전하가 따로 흐르는 것을 어떻게 관측하는가? 다음 문서에서 다룬다.

---

🧩 **Principle** — 순수 스핀의 다체 얽힘에서 가우스 법칙·게이지 퍼텐셜·무질량 광자가 *창발* 한다 — 게이지 원리는 미시 공리가 아니라 다수가 만든 조직 원리일 수 있다.
🔬 **Boundary** — 슬레이브-입자 평균장은 게이지 요동을 평균적으로만 다뤄, 2+1D 컴팩트 $U(1)$ 의 모노폴 닫힘이나 실험적 식별의 모호함에서 깨진다.
🪜 **Emergence** — 위 레이어(standard-model·L6·L5)에서 같은 구조가 우주의 모든 게이지장을 string-net 큐비트 그물의 창발로 다시 쓰며, 스핀 아이스 자기 단극·열홀 측정으로 검증된다.

## 📝 연습문제

**1. (기초)** 분해 $c_{i\sigma}=b_i^\dagger f_{i\sigma}$ 가 국소 $U(1)$ 게이지 자유도를 가짐을 보여라. 즉 $f_{i\sigma}\to e^{i\theta_i}f_{i\sigma},\ b_i\to e^{i\theta_i}b_i$ 하에서 물리적 전자 $c_{i\sigma}$ 가 불변임을 직접 확인하라.

<details><summary>해설</summary>

변환을 대입하면
$$c_{i\sigma}=b_i^\dagger f_{i\sigma}\to (e^{i\theta_i}b_i)^\dagger(e^{i\theta_i}f_{i\sigma})=e^{-i\theta_i}b_i^\dagger\,e^{i\theta_i}f_{i\sigma}=b_i^\dagger f_{i\sigma}=c_{i\sigma}.$$
보손과 페르미온이 *같은* 위상으로 회전하므로 켤레와 상쇄되어 전자는 불변이다. 이 국소 위상은 물리에 영향을 주지 않는 중복 — 그 중복을 추적하는 보상장이 게이지장 $a_{ij}$ 다. 입자 수 제약 $b^\dagger b+\sum f^\dagger f=1$ 도 이 변환에 불변임을 확인하라(개수 연산자는 위상에 무관). $\square$
</details>

**2. (심화)** 평균장 해밀토니안 $H_{\text{MF}}=-t\sum_{\langle ij\rangle}(f_i^\dagger e^{ia_{ij}}f_j+\text{h.c.})$ 에서, 게이지 변환 $f_i\to e^{i\theta_i}f_i$ 가 $a_{ij}\to a_{ij}+\theta_i-\theta_j$ 를 요구함을 보이고, 닫힌 고리(plaquette)를 도는 선속 $\Phi=\sum_{\text{loop}}a_{ij}$ 가 게이지 불변임을 증명하라.

<details><summary>해설</summary>

호핑항: $f_i^\dagger e^{ia_{ij}}f_j\to e^{-i\theta_i}f_i^\dagger\,e^{ia_{ij}}\,e^{i\theta_j}f_j=f_i^\dagger\,e^{i(a_{ij}-\theta_i+\theta_j)}f_j$.
이것이 원래 형태 $f_i^\dagger e^{ia_{ij}'}f_j$ 와 같으려면 $a_{ij}'=a_{ij}-\theta_i+\theta_j$ ... 부호 규약에 따라 변환은 $a_{ij}\to a_{ij}+\theta_i-\theta_j$ (켤레항 포함 일관성). 닫힌 고리 $i\to j\to k\to\cdots\to i$ 를 따라 합하면
$$\Phi\to\Phi+\sum_{\text{loop}}(\theta_i-\theta_j)=\Phi+0,$$
각 사이트의 $\theta$ 가 들어오고 나가며 상쇄되어 $\Phi$ 는 불변. 실험 1의 Wilson 선속이 이를 수치로 확인한다. 선속은 물리적 자기장(베리 곡률 유사)에 해당한다. $\square$
</details>

**3. (논문 비평)** Han et al. (Nature 2012)은 헤르베르츠마이트의 중성자 산란에서 *연속* 스펙트럼을 스피논 분수화의 증거로 해석했다. 이 해석에 대해 (a) 연속 스펙트럼이 분수화 없이 설명될 다른 가능성, (b) 게이지장의 닫힘/풀림을 직접 가르는 추가 측정을 비판적으로 논하라.

<details><summary>해설</summary>

(a) 대안: ① 결정 무질서·화학적 치환(Zn/Cu 사이트 혼합)이 들뜸을 비균질하게 퍼뜨려 연속처럼 보이게 할 수 있다. ② 강한 마그논-마그논 상호작용이나 다중-마그논 연속체도 넓은 스펙트럼을 낸다. 따라서 연속체 자체는 *필요* 조건이지 *충분* 조건이 아니다.

(b) 결정적 추가 측정: ① **열홀 전도도** — 게이지선속(vison)·마요라나의 양자화된 기여를 검출(키타에프 계열). ② **위상 얽힘 엔트로피** $\gamma=\ln\mathcal{D}$ — DMRG/실험적 추정으로 게이지 군 식별. ③ **자기장 의존성** — 풀림 상이면 들뜸 갭·연속체가 게이지선속 밀도에 따라 특징적으로 변함. ④ 토러스 기하 바닥상태 겹침. 핵심 비판: 자기 질서의 *부재* 와 연속체의 *존재* 는 정황 증거일 뿐, 게이지장의 풀림(deconfinement)을 직접 보이는 양성 신호가 필요하다는 점이다. $\square$
</details>
