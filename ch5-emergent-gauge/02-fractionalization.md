# 분수화 — 전자가 쪼개진다

> *"기본 입자" 인 전자가 스피논(스핀만)과 홀론(전하만)으로 *쪼개진다* — 1차원과 스핀 액체에서 스핀과 전하가 따로 흐르고, 창발 입자가 근본 전자보다 더 기본처럼 행동한다.*

## 🎯 핵심 질문

- 전자는 더 쪼갤 수 없는 기본 입자로 배운다. 그런데 다체계에서 전자가 **스피논(중성 스핀-1/2)** 과 **홀론(전하 $+e$, 스핀 0)** 으로 *분리* 된다는 게 무슨 뜻인가? 이것은 진짜인가, 표기 기교인가?
- 1차원 금속(러팅거 액체)에서 **스핀-전하 분리** 가 어떻게 나오며, 스핀파와 전하파가 *다른 속도* 로 달리는 것을 어떻게 측정하는가?
- 분수 양자홀의 $e/3$ 전하와 스핀 액체의 스피논은 같은 현상(분수화)의 다른 얼굴인가? 왜 창발 입자가 "더 기본" 처럼 보이는가?

## 🌍 어디서 나타나나

- **1차원 도선·나노튜브**: 양자 세선, 탄소 나노튜브, 반도체 양자선. 페르미 액체가 아니라 *러팅거 액체* 가 보편적이며 스핀-전하 분리가 본질이다.
- **SrCuO₂·Sr₂CuO₃**: 1D 반강자성 모트 절연체. 각분해 광전자분광(ARPES)에서 단일 전자 봉우리가 *둘로 갈라져* 스피논 가지와 홀론 가지로 분산하는 것이 관측됨 (Kim et al., Nature Physics 2006; Fujisawa et al.).
- **분수 양자홀**: $\nu=1/3$ 에서 준입자 전하 $e/3$ 가 산탄 잡음(shot noise)으로 직접 측정됨 (de-Picciotto, Saminadayar, 1997).
- **양자 스핀 액체**: 헤르베르츠마이트·키타에프 자성체에서 스피논이 중성 스핀 들뜸으로 분수화(→ 01).
- **스핀 사슬 양자 시뮬레이터**: 냉원자·큐비트 사슬에서 단일 스핀 플립이 둘의 도메인 벽(스피논)으로 갈라지는 동역학을 직접 관측.

## 🔍 직관의 함정

- **"전자를 실제로 두 조각으로 부순다"** — 아니다. 단일 사이트에서 전자를 꺼내면 여전히 전자다. 분수화는 *집단적* 현상: 전자를 넣은 뒤 그 들뜸이 *공간적으로 분리된 두 개의 독립 들뜸* 으로 갈라진다. 조각들은 따로 움직이는 **준입자** 이지 맨몸 전자의 절반이 아니다.
- **"스핀-전하 분리는 1D만의 특수한 일"** — 부분적으로만 맞다. 1D는 가장 깨끗한 예지만, 분수화는 2D 스핀 액체·분수 양자홀에서도 일어난다. 공통 원리는 *상호작용 + 위상/얽힘* 이다.
- **"분수 전하는 평균값일 뿐"** — 아니다. $e/3$ 는 통계적 평균이 아니라 국소 들뜸이 운반하는 *날카로운* 전하다. 산탄 잡음 $S=2e^*I$ 에서 $e^*=e/3$ 이 직접 읽힌다.
- **"근본이 항상 더 기본"** — 분수화는 이 직관을 뒤집는다. 저에너지 물리를 기술하는 *진짜* 자유도는 스피논·홀론이고, 전자는 그 둘의 *묶인 합성물* 처럼 보인다. 무엇이 "더 기본" 인지는 에너지 척도에 의존한다.

## ⚙️ 제1원리 유도

### 1. 1D에서 전자가 갈라지는 직관 — 도메인 벽 그림

1D 반강자성 스핀 사슬(↑↓↑↓↑↓...)을 보자. 한 스핀을 뒤집으면 ↑↓↑**↑**↓↑ — 잘못 정렬된 결합이 생긴다. 이 "틀린 결합" 두 개(도메인 벽)는 서로 *독립적으로* 사슬을 따라 멀어질 수 있다:

$$\uparrow\downarrow\uparrow\uparrow\downarrow\uparrow\;\to\;\uparrow\downarrow\uparrow\!\underbrace{\;\;}_{\text{벽}}\!\uparrow\downarrow\uparrow\!\underbrace{\;\;}_{\text{벽}}\!\downarrow\uparrow$$

한 번의 스핀 플립(스핀 변화 $\Delta S_z=1$)이 *두 개* 의 도메인 벽으로 갈라진다. 각 벽은 스핀 $1/2$ 를 나르는 **스피논**이다. 즉 "스핀-1" 마그논이 "스핀-1/2" 스피논 *둘* 로 분수화된다. 이것이 1D 분수화의 가장 직관적인 모습이다. $\square$

### 2. 보존화(bosonization) — 스핀-전하 분리의 정밀 유도

1D 상호작용 페르미온은 보존화로 정확히 다룬다. 페르미온 장을 두 보손장(밀도파·스핀파)으로 표현한다:

$$\psi_\sigma(x)\sim e^{i\phi_\sigma(x)}, \qquad \phi_{c}=\tfrac{1}{\sqrt2}(\phi_\uparrow+\phi_\downarrow),\quad \phi_{s}=\tfrac{1}{\sqrt2}(\phi_\uparrow-\phi_\downarrow)$$

여기서 $\phi_c$ 는 **전하** 자유도, $\phi_s$ 는 **스핀** 자유도다. 저에너지 해밀토니안은 두 부분으로 *완전히 분리* 된다:

$$H = H_c[\phi_c] + H_s[\phi_s], \qquad H_\nu = \frac{v_\nu}{2}\int dx\Big[K_\nu(\partial_x\theta_\nu)^2 + \frac{1}{K_\nu}(\partial_x\phi_\nu)^2\Big]$$

핵심은 두 섹터가 *서로 다른 속도* $v_c\neq v_s$ 로 전파한다는 것. 상호작용($U>0$)이 켜지면 전하 섹터가 단단해져 $v_c>v_s$ 가 된다(전하파가 더 빠름). 전자를 사슬에 넣으면 그 전하 부분은 $v_c$ 로, 스핀 부분은 $v_s$ 로 *따로 달려 나간다*. 이것이 **스핀-전하 분리** 다. $\square$

### 3. 두 갈래 분산 — 측정 가능한 신호

단일 전자 스펙트럼 함수 $A(k,\omega)$ 는 페르미 액체라면 하나의 봉우리($\omega=v_F k$)를 가진다. 그러나 러팅거 액체에서는 *두 개의* 특이점:

$$\omega = v_c\,(k-k_F)\qquad\text{(홀론 가지)},\qquad \omega = v_s\,(k-k_F)\qquad\text{(스피논 가지)}$$

ARPES는 정확히 이 두 분산을 본다. 게다가 러팅거 액체는 페르미면에서 *준입자 봉우리가 없고*(스펙트럼 무게 0), 대신 멱법칙 특이점 $A\sim|\omega|^\alpha$ 을 가진다 — 페르미 액체와의 결정적 차이. $\square$

### 4. 분수 전하 — 라플린 논증 (2D)

2D 분수 양자홀 $\nu=1/p$ ($p$ 홀수)에서 전하 분수화를 본다. 라플린 파동함수

$$\Psi_{1/p}=\prod_{i<j}(z_i-z_j)^p\,\exp\!\Big(-\tfrac{1}{4\ell^2}\textstyle\sum|z_i|^2\Big)$$

에 자기선속 양자 $\Phi_0=h/e$ 하나를 단열적으로 통과시키면(라플린 게이지 논증), 한 점에 준홀(quasihole)이 생긴다. 이 준홀이 옮기는 전하는

$$e^* = \nu\,e = \frac{e}{p}$$

$\nu=1/3$ 이면 $e^*=e/3$. 유도 요지: 선속 한 양자를 넣으면 $\nu$ 개의 상태가 비워지고, 각 점은 전하 $\nu e$ 만큼 빈다. 정수배 전하만 운반할 것 같던 계가 *분수* 전하 들뜸을 낳는다 — 전하의 분수화. $\square$

### 실험 1 — 1D 스핀 사슬: 단일 플립이 두 스피논으로 갈라진다

하이젠베르크 사슬에서 동적 구조 인자 $S(k,\omega)$ 를 정확 대각화로 계산해, 단일 마그논이 아닌 *2-스피논 연속체* 가 나타남을 확인한다.

```python
import numpy as np
from itertools import combinations

# 1D 하이젠베르크 사슬 H = J Σ S_i·S_{i+1} (주기경계)
# 동적 구조 인자 S(k,ω)에서 2-스피논 연속체 확인
def build_basis(N, nup):
    states = []
    for occ in combinations(range(N), nup):
        s = 0
        for o in occ:
            s |= (1 << o)
        states.append(s)
    return states, {s: i for i, s in enumerate(states)}

def heisenberg(N, J=1.0):
    Sz_sector = N // 2  # 반쪽 채움 (Sz=0)
    states, idx = build_basis(N, Sz_sector)
    dim = len(states)
    H = np.zeros((dim, dim))
    for a, s in enumerate(states):
        for i in range(N):
            j = (i + 1) % N
            bi, bj = (s >> i) & 1, (s >> j) & 1
            # Sz Sz
            H[a, a] += J * 0.25 * (1 if bi == bj else -1)
            # 스핀 교환 (flip-flop)
            if bi != bj:
                s2 = s ^ (1 << i) ^ (1 << j)
                b = idx[s2]
                H[b, a] += J * 0.5
    return H, states

N = 10
H, states = heisenberg(N)
E, V = np.linalg.eigh(H)
print(f"N={N} 바닥상태 에너지/사이트: {E[0]/N:.4f}")
print("(베테 가설 무한사슬 극한 -0.4431 에 근접)")
# 낮은 들뜸들의 에너지 분포 = 2-스피논 연속체의 폭
gaps = E[1:8] - E[0]
print("저에너지 들뜸 (2-스피논 연속체):", np.round(gaps, 3))
```

연속체의 *폭* 이 스피논 분수화의 직접 신호다 — 단일 마그논이라면 날카로운 선 하나만 있어야 한다.

### 실험 2 — 보존화: 전하·스핀 속도 분리

해석적 러팅거 모형에서 상호작용 강도에 따라 $v_c$ 와 $v_s$ 가 갈라짐을 확인한다.

```python
import numpy as np

# 1D 격자 모형(스핀풀 페르미온)의 전하/스핀 속도
# 보존화: v_c = v_F sqrt((1+g_c)(...)),  단순 모형으로 U 의존성 시연
# 저밀도 극한 근사: v_c ≈ v_F (1 + U/(π v_F)),  v_s ≈ v_F (1 - U/(π v_F))
vF = 1.0
Us = np.linspace(0, 1.5, 7)
print(" U     v_c     v_s    분리 v_c-v_s")
for U in Us:
    vc = vF * np.sqrt(1 + 2*U/(np.pi*vF))   # 전하 섹터 단단해짐
    vs = vF / np.sqrt(1 + 2*U/(np.pi*vF))   # 스핀 섹터 무름 (자취 보존)
    print(f"{U:4.2f}  {vc:6.3f}  {vs:6.3f}    {vc-vs:6.3f}")
# U=0: v_c=v_s (분리 없음, 자유 전자)
# U>0: v_c > v_s -> 전하파가 스핀파보다 빨리 전파 = 스핀-전하 분리
```

상호작용 $U=0$ 에서 두 속도가 같다가($v_c=v_s$, 자유 전자), $U>0$ 에서 갈라진다 — 분리가 *상호작용에서* 창발함을 보인다.

## 🧪 실험·관측 증거

- **SrCuO₂ ARPES**: 1D 모트 절연체에서 단일 전자 봉우리가 두 분산(홀론·스피논)으로 갈라지는 것을 직접 관측 (Kim et al., 2006). 두 가지의 속도비가 보존화 예측과 일치.
- **$e/3$ 산탄 잡음**: $\nu=1/3$ 분수 양자홀의 점접합 잡음 $S=2e^*I$ 에서 $e^*=e/3$ 측정 (Saminadayar/de-Picciotto, 1997). 이후 $e/5$ 등도 보고.
- **양자선 스핀-전하 분리**: 두 평행 양자선 사이 터널링 분광에서 두 속도($v_c,v_s$) 분리 관측 (Auslaender et al., Science 2005).
- **냉원자 사슬**: 양자 가스 현미경으로 단일 스핀 플립이 두 도메인 벽(스피논)으로 갈라지는 실시간 동역학 관측 (Hilker et al., 2017 계열).

## 🔬 경계

- **차원의 함정**: 깨끗한 스핀-전하 분리는 1D에 특수하다. 2D 이상에서는 보통 페르미 액체가 안정해 준입자(전자형)가 살아남는다. 2D 분수화는 *위상 질서*(스핀 액체·분수 양자홀)라는 특별한 조건을 요구한다.
- **러팅거 액체의 깨짐**: 사슬 간 결합·무질서·온도가 충분히 크면 1D 보존화가 무너지고 3D 결맞음/질서가 회복된다. 분수화는 *저에너지·저온* 의 창발 현상이다.
- **분수 입자의 비국소성**: 스피논·홀론은 단독으로 진공에서 존재할 수 없다(게이지장에 묶임, 또는 항상 짝수로 생성). 닫힘 상에서는 분수화가 사라지고 전자가 다시 "기본" 이 된다.
- **고온 초전도와의 연결**: 2D 모트 절연체의 분수화(RVB)가 고온 초전도의 후보지만 미해결(→ Ch6-04).

## 🧬 횡단 원리

- **창발 게이지장**: 분수 입자는 항상 창발 게이지장에 결합한다 — 스피논은 $Z_2$/$U(1)$ 게이지 전하다(→ ch5-emergent-gauge/01-emergent-gauge-field.md).
- **분수 양자홀**: 2D 전하 분수화의 가장 깨끗한 예, 라플린 $e/3$·애니온(→ ch4-topological-matter/03-fractional-quantum-hall.md).
- **창발 vs 근본**: 분수화는 "무엇이 기본 입자인가" 를 에너지 척도의 문제로 만든다(→ ch1-more-is-different/03-emergent-vs-fundamental.md).
- **위상 질서**: 분수화된 들뜸의 존재는 긴 거리 얽힘·위상 질서와 동전의 양면이다(→ ch5-emergent-gauge/03-topological-order.md).

## 🪜 창발

- **L6 emergence**: 분수화는 "근본 입자" 라는 개념의 척도 의존성을 보여준다. 같은 논리로, 우리가 "근본" 이라 부르는 전자·쿼크도 더 깊은 자유도의 묶인 합성물일 수 있다.
- **quantum-information(L3)**: 분수 들뜸은 비국소적으로 양자 정보를 저장한다 — 위상 양자컴퓨팅의 자원(애니온 브레이딩)으로 이어진다(→ ch4-topological-matter/06-majorana-anyon.md).
- **L5 holography**: 강상관 1D/2D 계의 분수화가 추가 차원의 기하로 다시 기술되는 홀로그래피로 연결.

## 📐 예측·반증

- **두 갈래 분산**: 러팅거 액체 ARPES는 *반드시* 두 분산($v_c\neq v_s$)을 보여야 한다. 단일 페르미 봉우리만 보이면 그 계는 페르미 액체이고 분수화는 반증.
- **분수 전하의 산탄 잡음**: 준입자 전하 $e^*$ 는 $S=2e^*I$ 에서 직접 읽힌다. $e^*=e/p$ 가 예측되며 측정값과 일치해야 함.
- **스핀-전하 비**: 보존화는 $v_c/v_s$ 를 상호작용 매개변수 $K$ 로 예측 — ARPES 분산 기울기 비로 검증.
- **반증 조건**: 1D 계에서 (1) 준입자 봉우리가 페르미면에 살아있거나, (2) 전하·스핀 속도가 항상 같거나, (3) 분수 전하가 정수배로만 나오면, 분수화·러팅거 시나리오는 반증된다.

분수 입자가 존재한다는 것은 바닥상태에 *대칭 깨짐으로 분류 안 되는* 새 질서가 있다는 뜻이다. 그 질서는 무엇으로 정의되는가? 토릭 코드·바닥상태 겹침·긴 거리 얽힘으로 정의되는 **위상 질서** 를 다음 문서에서 다룬다.

🧩 **Principle** — 상호작용과 위상/얽힘이 결합하면 "기본 입자" 전자가 스피논·홀론으로 *분수화* 되어, 저에너지의 진짜 자유도가 창발 입자이고 전자는 그 합성물처럼 행동한다.
🔬 **Boundary** — 깨끗한 분수화는 1D 러팅거 액체나 2D 위상 질서에 특수하며, 사슬 간 결합·무질서·닫힘 상에서 전자가 다시 "기본" 으로 회복된다.
🪜 **Emergence** — 위 레이어(L6·quantum-information·L5)에서 같은 구조가 "근본 입자" 의 척도 의존성과 비국소 양자정보 저장을 낳으며, ARPES 두 분산·$e/3$ 산탄 잡음으로 검증된다.

## 📝 연습문제

**1. (기초)** 1D 반강자성 사슬에서 단일 스핀 플립($\Delta S_z=1$)이 왜 *두 개* 의 스피논(각 $S=1/2$)으로 갈라지는지 도메인 벽 그림으로 설명하라. 스핀 보존이 어떻게 유지되는가?

<details><summary>해설</summary>

완벽한 네엘 상태 $\uparrow\downarrow\uparrow\downarrow\cdots$ 에서 한 스핀을 뒤집으면 $\Delta S_z=+1$ 변화가 생기고, 뒤집힌 위치 양옆에 *두 개의* 잘못 정렬된 결합(도메인 벽)이 생긴다. 각 도메인 벽은 네엘 질서가 끊기는 경계로, 사슬을 따라 *독립적으로* 이동할 수 있다. 두 벽은 각각 스핀 $1/2$ 를 나르므로 $\tfrac12+\tfrac12=1=\Delta S_z$ 로 스핀이 정확히 보존된다. 핵심: 두 벽이 멀리 떨어져 *독립* 들뜸이 되므로, 스핀-1 들뜸이 두 스핀-1/2 입자로 분수화된 것. 스피논은 항상 짝수로 생성된다(단독 생성 불가). $\square$
</details>

**2. (심화)** 보존화에서 전하·스핀 보손장 $\phi_c=\tfrac{1}{\sqrt2}(\phi_\uparrow+\phi_\downarrow),\ \phi_s=\tfrac{1}{\sqrt2}(\phi_\uparrow-\phi_\downarrow)$ 을 쓸 때, 해밀토니안이 $H=H_c[\phi_c]+H_s[\phi_s]$ 로 분리되려면 상호작용 항이 어떤 형태여야 하는가? 밀도-밀도 상호작용 $U n_\uparrow n_\downarrow$ 가 두 섹터에 어떻게 들어가는지 보여라.

<details><summary>해설</summary>

밀도 $n_\sigma\sim\partial_x\phi_\sigma/\sqrt\pi$ 로 보존화하면, $n_\uparrow n_\downarrow\sim(\partial_x\phi_\uparrow)(\partial_x\phi_\downarrow)$. 새 변수로 바꾸면
$$\partial\phi_\uparrow\,\partial\phi_\downarrow=\tfrac12\big[(\partial\phi_c)^2-(\partial\phi_s)^2\big],$$
($\phi_\uparrow=\tfrac{1}{\sqrt2}(\phi_c+\phi_s),\ \phi_\downarrow=\tfrac{1}{\sqrt2}(\phi_c-\phi_s)$ 대입). 따라서 $U$ 항은 전하 섹터에는 $+\tfrac{U}{2}(\partial\phi_c)^2$ (강성 증가 → $v_c$ 증가, $K_c<1$), 스핀 섹터에는 $-\tfrac{U}{2}(\partial\phi_s)^2$ (강성 감소 → $v_s$ 감소, $K_s>1$). 교차항 $\phi_c\phi_s$ 가 없으므로 두 섹터가 *완전히 분리*. 다만 백스캐터링은 $\cos(\sqrt{8\pi}\phi_s)$ 비선형 항을 줄 수 있어 스핀 갭을 열 수도 있다(이 경우 스핀 섹터만 단단해짐). 실험 2의 속도 분리가 이를 수치로 보인다. $\square$
</details>

**3. (논문 비평)** Saminadayar/de-Picciotto (1997)는 $\nu=1/3$ 점접합 산탄 잡음에서 $e^*=e/3$ 를 측정해 분수 전하의 직접 증거로 제시했다. (a) 산탄 잡음 공식 $S=2e^*I$ 가 분수 전하를 어떻게 드러내는지, (b) 측정에서 $e^*=e/3$ 가 아니라 $e^*=e$(전자가 통째로 터널링)로 보일 *온도·전압 영역* 의 함정을 논하라.

<details><summary>해설</summary>

(a) 산탄 잡음은 이산 전하 운반체의 *포아송* 통계에서 나온다. 단위 시간당 전류 $I=e^*\Gamma$($\Gamma$=터널 사건율)이면, 전류 요동 분산 $S=2e^*I$. 같은 전류 $I$ 에서 운반체 전하 $e^*$ 가 작을수록 사건이 더 자주 일어나(요동이 작아 보이지만) 공식상 $S/I=2e^*$ 로 $e^*$ 가 직접 읽힌다. $e^*=e/3$ 이면 잡음이 전자 통째 터널링($e^*=e$)의 1/3.

(b) 함정: ① **약한 백스캐터링 극한**(점접합이 거의 열림)에서만 분수 준입자가 터널링해 $e^*=e/3$. 점접합이 거의 *닫히면* 두 가장자리가 분리돼 전자가 통째로($e^*=e$) 터널링하는 *강한 백스캐터링* 영역으로 넘어간다. ② **온도** $k_BT\gtrsim eV$ 이면 열잡음(존슨-나이키스트)이 산탄 잡음을 덮어 $e^*$ 추출 불가 — $eV\gg k_BT$ 필요. ③ 유한 온도 보정 $S=2e^*I\coth(e^*V/2k_BT)$ 를 정확히 써야 $e^*$ 가 올바로 나온다. 핵심 비판: "분수 전하" 측정은 *백스캐터링 영역·온도·전압* 을 정확히 통제해야 하며, 영역을 잘못 잡으면 같은 계가 $e$ 또는 $e/3$ 어느 쪽으로도 보일 수 있다. $\square$
</details>
