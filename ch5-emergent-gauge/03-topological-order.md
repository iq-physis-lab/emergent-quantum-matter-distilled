# 위상 질서 — 대칭 깨짐 없는 질서

> *토릭 코드의 바닥상태는 어떤 대칭도 깨지 않고 어떤 국소 질서변수도 없지만, *위상에 따라 겹친* 바닥상태와 긴 거리 얽힘으로 란다우 패러다임 너머의 새 질서를 정의한다.*

## 🎯 핵심 질문

- 란다우의 질서 패러다임은 모든 상(phase)을 *대칭 깨짐* 과 *국소 질서변수* 로 분류한다. 그런데 어떤 대칭도 깨지지 않고 국소 질서변수도 없는데 *서로 다른 상* 이 있다면 — 무엇이 그들을 가르는가?
- **토릭 코드**(키타에프)의 바닥상태가 *토러스에서 4겹으로 겹친다* 는 것은 무슨 뜻이고, 이 겹침이 왜 국소 섭동에 무관하게 보호되는가?
- "긴 거리 얽힘(long-range entanglement)" 과 애니온 들뜸은 위상 질서의 정의에 어떻게 들어가는가?

## 🌍 어디서 나타나나

- **분수 양자홀 상태**: 위상 질서의 첫 물리적 실현. $\nu=1/3$ 라플린 상태는 토러스에서 3겹 바닥상태, 분수 전하·애니온을 가진다.
- **양자 스핀 액체**: $Z_2$ 스핀 액체(키타에프 허니컴, RVB)는 토릭 코드와 같은 위상 질서 부류. 헤르베르츠마이트·$\alpha$-RuCl₃ 후보.
- **위상 양자 메모리**: 토릭 코드는 *양자 오류정정 부호* 그 자체다. 바닥상태 겹침이 논리 큐비트를 비국소적으로 저장 → 표면 부호(surface code).
- **양자 시뮬레이터**: 초전도 큐비트(Google, 2021)·중성원자 격자에서 토릭 코드 바닥상태를 준비하고 애니온을 생성·브레이딩.

## 🔍 직관의 함정

- **"위상(topology) = 모양(geometry)"** — 아니다. 위상 질서의 "위상" 은 도넛 구멍 같은 공간 모양이 아니라 *바닥상태 파동함수의 위상적 성질*(긴 거리 얽힘, 애니온 융합 규칙)이다. 같은 모양의 격자에서 위상 질서가 있을 수도, 없을 수도 있다.
- **"질서는 항상 대칭 깨짐"** — 위상 질서가 이 도그마를 깬다. 토릭 코드 바닥상태는 대칭을 *깨지 않는다*(어떤 국소 질서변수의 기댓값도 0). 그래도 보통 곱상태(product state)와 *다른 상* 이다.
- **"바닥상태 겹침은 우연한 대칭 때문"** — 아니다. 위상적 겹침은 우연이 아니라 *위상 보호* 된다. 어떤 국소 연산자도 겹친 바닥상태들을 구별하거나 섞을 수 없다 — 그래서 양자 메모리가 된다.
- **"애니온은 이론적 장난감"** — 아니다. 토릭 코드의 $e$(전기)·$m$(자기) 애니온은 융합·브레이딩 규칙을 가진 실제 들뜸이고, 큐비트 실험에서 생성·이동된다.

## ⚙️ 제1원리 유도

### 1. 토릭 코드 해밀토니안과 스타빌라이저

2D 격자(주기경계=토러스)의 각 *링크* 에 큐비트 하나를 놓는다. 두 종류의 연산자를 정의한다:

- **꼭짓점(star) 연산자**: 꼭짓점 $s$ 에 닿는 4개 링크의 $\prod X$
$$A_s = \prod_{i\in s} X_i$$
- **면(plaquette) 연산자**: 면 $p$ 를 둘러싼 4개 링크의 $\prod Z$
$$B_p = \prod_{i\in p} Z_i$$

키타에프 토릭 코드 해밀토니안:

$$H = -\sum_s A_s - \sum_p B_p$$

모든 $A_s,B_p$ 는 서로 *교환* 한다(임의의 star와 plaquette는 0개 또는 2개의 공통 링크를 공유 → $XZ$ 가 짝수 번 → 교환). 게다가 $A_s^2=B_p^2=1$ 이므로 고윳값 $\pm1$. 바닥상태는 모든 $A_s=+1,\,B_p=+1$ 인 상태다. 이런 *서로 교환하는 안정자(stabilizer)* 의 동시 고유상태로 정의되는 것이 토릭 코드다. $\square$

### 2. 바닥상태 겹침 — 위상에 의존하는 차원 세기

바닥상태의 *수* 를 세자. 큐비트 수 $=$ 링크 수 $N_\ell$. 독립 제약 조건 수를 세면:

- star 연산자: $\prod_s A_s = 1$ (각 링크가 두 꼭짓점에 공유 → 모든 $X$ 가 짝수 번) → 독립 star는 $N_v-1$ 개.
- plaquette 연산자: $\prod_p B_p = 1$ 마찬가지 → 독립 plaquette는 $N_f-1$ 개.

바닥상태 공간 차원:

$$\log_2(\text{겹침}) = N_\ell - (N_v-1) - (N_f-1) = (N_\ell - N_v - N_f) + 2 = -\chi + 2$$

여기서 $\chi=N_v-N_\ell+N_f$ 는 표면의 **오일러 지표**. 종수 $g$ 의 닫힌 곡면은 $\chi=2-2g$ 이므로:

$$\log_2(\text{겹침}) = 2g \quad\Rightarrow\quad \text{바닥상태 겹침} = 4^g$$

토러스($g=1$)는 **4겹**, 구($g=0$)는 1겹. 바닥상태 수가 *공간의 위상(종수)* 에만 의존한다 — 이것이 위상 질서의 정의적 특징이다. 국소 물리는 전혀 모르고 전역 위상만 안다. $\square$

### 3. 애니온 들뜸 — 비가환 끈 연산자

바닥상태 위에 들뜸을 만들자. 하나의 $B_p=-1$ 로 만드는 들뜸이 **전기 전하 $e$**(꼭짓점형), 하나의 $A_s=-1$ 이 **자기 선속 $m$**(면형) 애니온이다. 이들은 *끈(string) 연산자* 의 끝점에서 쌍으로 생긴다:

$$S^e_{\gamma}=\prod_{i\in\gamma}Z_i \quad(\text{끝점에 }e), \qquad S^m_{\tilde\gamma}=\prod_{i\in\tilde\gamma}X_i\quad(\text{끝점에 }m)$$

$e$ 와 $m$ 끈이 교차할 때 한 번의 반교환($XZ=-ZX$)이 일어나, $e$ 가 $m$ 을 한 바퀴 도는 브레이딩은 위상 $-1$ 을 준다:

$$|e\,\text{가}\,m\,\text{을 감음}\rangle = -|\text{원래}\rangle$$

$e,m$ 은 각각 보손이지만 서로에 대해 *상호 준통계*($-1$)를 갖는 애니온이다. 둘의 복합체 $\varepsilon=e\times m$ 은 페르미온. 융합 규칙 $e\times e=1,\ m\times m=1,\ e\times m=\varepsilon$ 이 토릭 코드의 위상 데이터를 완성한다. $\square$

### 4. 긴 거리 얽힘 — 끈 연산자가 바닥상태를 잇는다

토러스에서 4개의 바닥상태는 *공간을 감는*(non-contractible) 끈 연산자로 서로 연결된다:

$$W_x=\prod_{i\in C_x}Z_i, \qquad W_y=\prod_{i\in C_y}X_i$$

($C_x,C_y$ 는 토러스의 두 비축약 고리). 이들은 모든 안정자와 교환하지만(바닥상태 공간 보존) 서로는 *반교환* $W_xW_y=-W_yW_x$ 한다. 이 비가환 대수의 최소 표현이 정확히 $2\times2$(한 축당) → 두 축 합쳐 $4$겹 바닥상태. 핵심: 이 끈은 *비국소* 연산자다. 어떤 *국소* 연산자도 바닥상태들을 구별/연결할 수 없다. 그래서 바닥상태들은 모든 국소 관측으로 *구별 불가능* 하면서도 비국소적으로 다르다 — 이것이 **긴 거리 얽힘** 이고, 어떤 유한 깊이 국소 양자회로로도 곱상태로 풀 수 없는 얽힘이다. $\square$

### 실험 1 — 토릭 코드 바닥상태 겹침: 스타빌라이저 직접 구성

작은 토러스(2×2 격자, 8 큐비트)에서 모든 스타빌라이저를 만들고 $A_s=B_p=+1$ 부분공간의 차원이 4임을 확인한다.

```python
import numpy as np
from functools import reduce

# 2x2 주기격자 토릭 코드: 링크 8개 = 큐비트 8개
# 꼭짓점 4개, 면 4개 (토러스)
I = np.eye(2); X = np.array([[0,1],[1,0]]); Z = np.array([[1,0],[0,-1]])

def op_on(paulis, n=8):
    return reduce(np.kron, paulis)

# 링크 인덱스: 2x2 격자에서 수평링크 4 + 수직링크 4
# 꼭짓점 (i,j)에 닿는 링크: 좌/우 수평, 상/하 수직 (주기경계)
# 단순화를 위해 표준 토릭코드 배선 사용
# 수평링크 h[i][j], 수직링크 v[i][j], i,j in {0,1}
def hidx(i,j): return 2*i + j          # 0..3
def vidx(i,j): return 4 + 2*i + j      # 4..7

def star(i, j):  # 꼭짓점 (i,j) 의 A_s = ∏X (닿는 4 링크)
    links = [hidx(i,j), hidx(i,(j-1)%2), vidx(i,j), vidx((i-1)%2,j)]
    p = [I]*8
    for l in links: p[l] = X
    return op_on(p)

def plaq(i, j):  # 면 (i,j) 의 B_p = ∏Z (둘러싼 4 링크)
    links = [hidx(i,j), hidx((i+1)%2,j), vidx(i,j), vidx(i,(j+1)%2)]
    p = [I]*8
    for l in links: p[l] = Z
    return op_on(p)

H = np.zeros((256,256))
for i in range(2):
    for j in range(2):
        H = H - star(i,j) - plaq(i,j)

E = np.linalg.eigvalsh(H)
E0 = E[0]
deg = np.sum(np.abs(E - E0) < 1e-9)
print(f"바닥상태 에너지 = {E0:.3f}")
print(f"바닥상태 겹침 = {deg}  (토러스 g=1 -> 4^1 = 4 예측)")
# 들뜸 갭 (애니온 쌍 생성 비용)
gap = E[E > E0 + 1e-9][0] - E0
print(f"들뜸 갭 = {gap:.3f}")
```

바닥상태 겹침 = 4 가 종수 1 토러스의 $4^g=4$ 예측과 정확히 일치 — 위상 질서의 직접 증거다.

### 실험 2 — 비축약 끈 연산자: 바닥상태를 잇는 비가환 대수

토러스를 감는 두 끈 $W_x,W_y$ 가 반교환함을 확인하고, 이것이 4겹 겹침을 강제함을 본다.

```python
import numpy as np
from functools import reduce
I = np.eye(2); X = np.array([[0,1],[1,0]]); Z = np.array([[1,0],[0,-1]])
def op_on(paulis): return reduce(np.kron, paulis)
def hidx(i,j): return 2*i+j
def vidx(i,j): return 4+2*i+j

# W_x: x 방향을 감는 Z 끈 (수평 고리의 수직링크들)
def Wz_loop():
    p=[I]*8
    for j in range(2): p[vidx(0,j)] = Z   # 한 행을 감는 비축약 고리
    return op_on(p)
# W_y: y 방향을 감는 X 끈
def Wx_loop():
    p=[I]*8
    for i in range(2): p[hidx(i,0)] = X
    return op_on(p)

Wz, Wx = Wz_loop(), Wx_loop()
comm = Wz @ Wx - Wx @ Wz
anti = Wz @ Wx + Wx @ Wz
print("교환자 [Wz,Wx] 노름:", np.linalg.norm(comm))
print("반교환자 {Wz,Wx} 노름:", np.linalg.norm(anti))
# 한 고리는 반교환 -> 비가환 대수 -> 바닥상태 부분공간에서 2겹씩
# 두 독립 비축약 방향 -> 2x2 = 4겹 위상 겹침
```

## 🧪 실험·관측 증거

- **분수 양자홀**: $\nu=1/3$ 의 토러스 3겹 바닥상태 겹침이 수치(정확 대각화)로 확인되고, 분수 전하·애니온 통계가 실험적으로 측정됨.
- **초전도 큐비트 토릭 코드**: Google(2021)·여러 그룹이 토릭 코드 바닥상태를 준비하고 $e,m$ 애니온을 생성·브레이딩해 $-1$ 위상을 관측.
- **위상 얽힘 엔트로피**: 수치(DMRG)로 키타에프·토릭 코드 모형에서 $\gamma=\ln2$($\mathcal{D}=2$)를 추출 — 위상 질서의 정량 신호(→ 04).
- **표면 부호 양자 메모리**: 위상 보호된 바닥상태 겹침이 논리 큐비트의 비국소 저장으로 활용, 오류정정 시연.

## 🔬 경계

- **유한 온도**: 2D 토릭 코드는 *유한 온도에서 위상 질서가 사라진다*(애니온이 열적으로 자유 생성, 양자 메모리 수명 유한). 안정한 위상 양자 메모리는 4D 또는 능동 오류정정이 필요.
- **위상 분류의 미완성**: 모든 위상 질서를 분류하는 완전한 수학(2D는 모듈러 텐서 범주로 잘 이해되지만, 3D 이상·대칭 보호 위상 포함은 진행 중)은 아직 미완.
- **국소 섭동의 한계**: 위상 보호는 *충분히 약한* 국소 섭동에만 성립한다. 섭동이 갭을 닫을 만큼 강하면 위상 상전이로 질서가 파괴된다.
- **실험적 식별**: 바닥상태 겹침은 토러스 기하를 요구해 실험적 직접 측정이 어렵다. 위상 얽힘 엔트로피·애니온 통계 같은 우회 신호에 의존.

## 🧬 횡단 원리

- **창발 게이지장**: 토릭 코드는 $Z_2$ 게이지 이론의 풀림 상이다. $e=Z_2$ 전하, $m=Z_2$ 선속(→ ch5-emergent-gauge/01-emergent-gauge-field.md).
- **분수화**: 애니온은 분수 통계를 나르는 분수 입자다(→ ch5-emergent-gauge/02-fractionalization.md).
- **얽힘**: 위상 질서 = 긴 거리 얽힘. 위상 얽힘 엔트로피 $\gamma=\ln\mathcal{D}$ 가 정량화한다(→ ch5-emergent-gauge/04-entanglement-phases.md).
- **위상 보호**: 바닥상태 겹침의 견고함은 위상 보호의 한 형태다(→ ch4-topological-matter/05-topological-protection.md).

## 🪜 창발

- **quantum-information(L3)**: 위상 질서는 *위상 양자 오류정정*(표면 부호)의 물리적 토대다. 비국소 얽힘이 논리 정보를 국소 노이즈로부터 보호.
- **entanglement-measurement(L3)**: 위상 질서를 얽힘 구조로 정량화하는 프로그램의 핵심 — 란다우 패러다임을 *얽힘 패러다임* 으로 대체.
- **L5 holography**: 긴 거리 얽힘과 기하의 관계가 홀로그래피로 이어진다. 얽힘이 시공간을 짓는다는 그림의 응집물질판.

## 📐 예측·반증

- **종수 의존 겹침**: 위상 질서는 종수 $g$ 곡면에서 $\mathcal{D}^{2g}$ (또는 토릭 코드 $4^g$) 겹침을 예측. 수치 정확 대각화로 검증(실험 1).
- **위상 얽힘 엔트로피**: $\gamma=\ln\mathcal{D}>0$ — 토릭 코드 $\ln2$. 곱상태·대칭 깨짐 상은 $\gamma=0$.
- **애니온 브레이딩 위상**: $e$ 가 $m$ 을 감으면 $-1$. 큐비트 실험으로 직접 측정.
- **반증 조건**: 어떤 상에서 (1) 토러스 바닥상태가 비축약 섭동에도 *유일* 하거나, (2) $\gamma=0$ 이거나, (3) 애니온 브레이딩 위상이 자명하면, 위상 질서는 그 상에서 반증된다.

위상 질서를 *바닥상태 겹침* 이나 *애니온* 없이, 단일 바닥상태의 **얽힘 구조** 만으로 진단할 수 있는가? 얽힘 엔트로피의 면적 법칙 $S\sim\alpha L-\gamma$ 와 위상항 $\gamma$ 가 어떻게 물질의 상을 규정하는지 다음 문서에서 다룬다.

---

🧩 **Principle** — 토릭 코드는 대칭을 깨지 않고 국소 질서변수도 없지만, 종수에 의존하는 바닥상태 겹침과 긴 거리 얽힘·애니온으로 란다우 너머의 위상 질서를 *창발* 시킨다.
🔬 **Boundary** — 2D 위상 질서는 유한 온도에서 사라지고, 강한 국소 섭동이 갭을 닫으면 위상 상전이로 파괴되며, 위상 분류의 완전한 수학은 미완이다.
🪜 **Emergence** — 위 레이어(quantum-information·entanglement·L5)에서 같은 구조가 표면 부호 양자 오류정정과 얽힘-기하 대응을 낳으며, $4^g$ 바닥상태 겹침·$\gamma=\ln2$·애니온 $-1$ 브레이딩으로 검증된다.

## 📝 연습문제

**1. (기초)** 토릭 코드에서 임의의 star 연산자 $A_s$ 와 plaquette 연산자 $B_p$ 가 항상 교환($[A_s,B_p]=0$)함을 보여라. (힌트: $A_s$ 와 $B_p$ 가 공유하는 링크 수를 세라.)

<details><summary>해설</summary>

$A_s=\prod X$, $B_p=\prod Z$. 두 연산자가 다른 링크에 작용하면 자명히 교환($X,Z$ 가 다른 큐비트). 같은 링크에서만 $XZ=-ZX$ 로 반교환 기여. $A_s$ 는 꼭짓점 $s$ 에 닿는 링크들, $B_p$ 는 면 $p$ 를 둘러싼 링크들에 작용한다. 격자 기하상 한 꼭짓점과 한 면은 *0개 또는 정확히 2개* 의 링크를 공유한다. 공유 링크가 0개면 교환. 2개면 각 공유 링크에서 $(-1)$ 씩 두 번 → $(-1)^2=+1$ → 교환. 따라서 모든 $A_s,B_p$ 가 교환하고, 동시 고유상태(바닥상태)가 잘 정의된다. 이것이 토릭 코드가 *정확히 풀리는* 이유다. $\square$
</details>

**2. (심화)** 종수 $g$ 닫힌 곡면 위 토릭 코드의 바닥상태 겹침이 $4^g$ 임을 오일러 지표로 유도하라. 큐비트 수(링크), 독립 제약 수를 세는 과정을 명시하라.

<details><summary>해설</summary>

큐비트 수 $=N_\ell$(링크). 안정자 $A_s$($N_v$개), $B_p$($N_f$개)는 각각 하나의 전역 관계 $\prod_s A_s=\prod_p B_p=1$ 을 가진다(각 링크가 정확히 두 꼭짓점/두 면에 공유 → 모든 $X$(또는 $Z$)가 짝수 번 등장 → 곱이 항등원). 따라서 독립 안정자 수 $=(N_v-1)+(N_f-1)$. 각 독립 안정자가 힐베르트 공간 차원을 절반으로 줄이므로
$$\log_2(\text{겹침})=N_\ell-(N_v-1)-(N_f-1)=-(N_v-N_\ell+N_f)+2=-\chi+2.$$
종수 $g$ 곡면 $\chi=2-2g$ 대입: $\log_2(\text{겹침})=-(2-2g)+2=2g$, 즉 겹침 $=2^{2g}=4^g$. 토러스 $g=1$ → 4겹(실험 1 확인). 일반 위상 질서는 $4^g$ 대신 $\mathcal{D}^{2g}$($\mathcal{D}$=전체 양자 차원). $\square$
</details>

**3. (논문 비평)** 키타에프(2003)의 토릭 코드는 *유한 온도에서 위상 질서가 불안정* 하다(2D). 그럼에도 표면 부호는 가장 유망한 실용 양자 오류정정 후보다. (a) 유한 온도 불안정성과 실용성이 어떻게 양립하는지, (b) "위상 보호 = 자동 보호" 라는 오해의 한계를 논하라.

<details><summary>해설</summary>

(a) 양립의 핵심은 *수동(passive) vs 능동(active) 보호* 의 구분이다. 2D 토릭 코드는 유한 온도에서 애니온이 자유 생성되어 *수동* 양자 메모리로는 수명이 유한(자기-정정 메모리가 아님). 그러나 표면 부호는 *능동* 오류정정을 쓴다: 안정자 $A_s,B_p$ 를 *반복 측정* 해 애니온(오류 신드롬)의 위치를 알아내고, 고전 디코더(예: 최소 가중 매칭)로 오류를 추적·정정한다. 위상 구조가 주는 것은 "자동 보호" 가 아니라 (i) 국소 오류가 *국소 신드롬* 으로 드러나 검출 가능, (ii) 논리 오류는 *전역 비축약 끈* 을 만들어야 하므로 거리 $d$ 에 지수적으로 억제, (iii) 높은 오류 문턱값(~1%). 즉 위상은 *능동 정정을 쉽게/효율적으로* 만든다.

(b) "위상 보호 = 자동" 오해의 한계: ① 유한 온도·유한 시간에서 애니온이 쌓이면 정정 없이는 논리 오류 발생. ② 위상 보호는 *충분히 약한 국소* 섭동에만 성립 — 상관된 오류·누설·측정 오류는 보호 밖. ③ 4D 토릭 코드는 자기-정정이 되지만 물리적 구현 불가. 핵심 비판: 위상 질서는 오류정정의 *강력한 토대* 이지 *마법적 면역* 이 아니다. 실용 양자컴퓨팅은 위상 구조 + 끊임없는 능동 정정의 결합을 요구한다. $\square$
</details>
