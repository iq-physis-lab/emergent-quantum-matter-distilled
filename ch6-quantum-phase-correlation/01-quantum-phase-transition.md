# 양자 상전이 — T=0 전이

> *온도가 0인데도 상이 바뀐다. 열요동이 모두 얼어붙은 곳에서, 양자 요동만으로 결합상수 $g$ 가 임계값을 넘는 순간 바닥상태 자체가 갈라진다 — $d$ 차원 양자계가 $d+z$ 차원 고전계로 사상되며.*

## 🎯 핵심 질문

- 열요동이 전혀 없는 **절대영도** 에서 어떻게 상전이가 일어나는가? 무엇이 "요동" 의 역할을 대신하는가?
- 결합상수 $g$ (자기장·압력·도핑)가 제어변수가 될 때, 바닥상태에 **비해석적 특이점** 이 생기는 조건은 무엇인가?
- 왜 $d$ 차원 양자 상전이가 $d+z$ 차원 **고전** 상전이와 같은가? 동적 임계지수 $z$ 는 어디서 오는가?

## 🌍 어디서 나타나나

양자 상전이(quantum phase transition, QPT)는 추상적 사고실험이 아니라 실험실의 다이얼을 돌리면 만나는 현실이다.

- **$\mathrm{LiHoF_4}$, $\mathrm{CoNb_2O_6}$**: 횡자기장을 걸어 강자성–상자성 전이를 $T=0$ 부근에서 통과시키는 횡자기장 이징 모형의 물질적 실현.
- **중후전자(heavy-fermion) 화합물** $\mathrm{CeCu_{6-x}Au_x}$, $\mathrm{YbRh_2Si_2}$: 도핑·압력·자기장으로 반강자성 양자 임계점을 통과 — 그 위에서 이상 금속(다음 다음 문서)이 펼쳐진다.
- **초전도–절연체 전이**: 박막의 무질서나 자기장을 제어변수로 한 $T=0$ 전이, 보편 저항 $\sim h/4e^2$.
- **냉원자 광격자**: $^{87}\mathrm{Rb}$ 보존을 광격자에 넣고 깊이를 바꿔 초유체–모트 전이를 직접 관측 — QPT를 "손으로 돌려" 본다.

공통점: 온도가 아니라 **비열적(non-thermal) 매개변수** 가 전이를 일으킨다.

## 🔍 직관의 함정

**함정 1 — "$T=0$ 이면 요동이 없으니 전이도 없다."** 고전적 직관이다. 고전 상전이는 $\exp(-E/k_BT)$ 의 열요동이 질서를 녹인다. $T\to 0$ 이면 이 요동은 사라진다. 그러나 **하이젠베르크 불확정성** 이 남는다. 운동량과 위치를 동시에 정할 수 없으므로 바닥상태조차 영점요동(zero-point fluctuation)을 갖는다. QPT를 일으키는 것은 바로 이 **양자 요동** 이다.

**함정 2 — "양자 상전이는 그냥 아주 낮은 온도의 고전 상전이다."** 아니다. 질적으로 다르다. 고전 전이의 제어변수는 $T$, 질서를 깨는 것은 열 엔트로피다. 양자 전이의 제어변수는 결합상수 $g$, 질서를 깨는 것은 운동에너지(비가환 항)다. 둘은 보편성류(universality class)도 다르다 — 시간 차원이 추가되기 때문이다.

**함정 3 — "양자 임계점은 $T=0$ 한 점이라 측정 불가능하다."** $T=0$ 은 실험적으로 닿을 수 없지만, 양자 임계점은 유한 온도 위로 **부채꼴 영역**(다음 문서)을 드리운다. 즉 $T>0$ 에서도 그 그림자를 측정한다 — 비열·저항·자화율의 비정상 거듭제곱 법칙으로.

## ⚙️ 제1원리 유도

### 1. 무엇이 바닥상태를 비해석적으로 만드는가

양자계의 해밀토니안을 두 비가환 부분으로 쓴다:

$$H(g) = H_0 + g\,H_1, \qquad [H_0, H_1] \neq 0.$$

$g=0$ 이면 $H_0$ 의 바닥상태(예: 모든 스핀 정렬)가 답이다. $g\to\infty$ 이면 $H_1$ 의 바닥상태(예: 모든 스핀이 횡자기장 방향)가 답이다. 두 극한의 바닥상태는 **서로 다른 질서** 를 가진다. 유한 $g$ 에서 바닥상태 에너지 $E_0(g)$ 와 파동함수 $|\psi_0(g)\rangle$ 가 $g$ 에 대해 매끄럽게 연속되면 전이는 없다. 그러나 어떤 임계값 $g_c$ 에서 $E_0(g)$ 가 **비해석적**(특이점)이 되면 — 그것이 양자 상전이다.

비해석성은 보통 **에너지 갭의 닫힘** 과 함께 온다. 바닥상태와 첫 들뜬상태 사이의 갭을 $\Delta(g)$ 라 하면, 유한계에서는 회피교차(avoided crossing)로 갭이 작아지지만 닫히지 않는다. 열역학 극한 $N\to\infty$ 에서 비로소

$$\Delta(g) \xrightarrow{N\to\infty} 0 \quad \text{at } g=g_c$$

가 되며, 이때 $E_0(g)$ 가 $g_c$ 에서 미분불가능해진다. $\square$

### 2. 모형: 1차원 횡자기장 이징

가장 깨끗한 예가 횡자기장 이징 모형(transverse-field Ising model, TFIM)이다.

$$H = -J\sum_{i} \sigma^z_i \sigma^z_{i+1} - h\sum_i \sigma^x_i.$$

여기서 $\sigma^z\sigma^z$ 항은 이웃 스핀을 $z$ 방향으로 정렬시키려 한다(강자성, 질서). $\sigma^x$ 항은 각 스핀을 $x$ 방향 고유상태로 돌리려 한다 — $z$ 기저에서는 $\sigma^x = |\!\uparrow\rangle\langle\downarrow| + |\!\downarrow\rangle\langle\uparrow|$ 로 스핀을 뒤집는 **양자 요동** 이다. 무차원 결합상수는 $g \equiv h/J$ 다.

- $g \ll 1$: 강자성. $\langle\sigma^z\rangle \neq 0$, 이중 축퇴 바닥상태(↑↑↑ 또는 ↓↓↓).
- $g \gg 1$: 상자성. 모든 스핀이 $+x$, $\langle\sigma^z\rangle = 0$, 유일한 바닥상태.

두 상은 다른 대칭(스핀 뒤집기 $\mathbb{Z}_2$ 의 깨짐 여부)을 가지므로 매끄럽게 이어질 수 없다 → 사이에 양자 임계점 $g_c$ 가 있다.

### 3. Jordan–Wigner로 정확히 푼다

TFIM은 정확히 풀린다. Jordan–Wigner 변환으로 스핀을 무상호작용 페르미온으로 사상한다:

$$\sigma^z_i = 1 - 2c_i^\dagger c_i, \qquad \sigma^x_i = (c_i^\dagger + c_i)\prod_{j<i}(1-2c_j^\dagger c_j).$$

해밀토니안은 이차형식이 되고, Bogoliubov 변환 후 자유 페르미온의 분산이 나온다:

$$\varepsilon_k = 2J\sqrt{1 + g^2 - 2g\cos k}.$$

들뜸 갭은 $\varepsilon_k$ 의 최솟값이다. $g<1$ 이면 $k=\pi$ 에서, $g>1$ 이면 $k=0$ 에서 최소가 되고, 두 경우 모두

$$\Delta = \min_k \varepsilon_k = 2J\,|1-g|.$$

$g=g_c=1$ 에서 **갭이 정확히 닫힌다**. $\square$ 임계점에서 분산은 $k\to 0$ 근처에서 $\varepsilon_k \approx 2J|k|$ 로 **선형** — 이것이 동적 임계지수 $z=1$ 의 신호다(에너지 $\sim$ 운동량$^z$).

상관길이는 $\xi \sim |g-g_c|^{-\nu}$ 로 발산하며 TFIM에서 $\nu=1$, 갭은 $\Delta \sim |g-g_c|^{z\nu}$ 로 닫힌다.

### 4. 양자–고전 대응: $d$ 차원 양자 ↔ $d+z$ 차원 고전

QPT의 가장 깊은 통찰이다. 분배함수를 허수시간 경로적분으로 쓴다:

$$Z = \mathrm{Tr}\,e^{-\beta H} = \int \mathcal{D}\phi\, e^{-S[\phi]}, \qquad \beta = 1/k_BT.$$

허수시간 $\tau$ 는 $0$ 에서 $\beta\hbar$ 까지 흐르고, 작용 $S$ 는 공간 $d$ 차원 + 시간 1차원에서의 적분이다. 즉 **$d$ 차원 양자계의 분배함수가 $(d+1)$ 차원 고전 통계계의 분배함수와 같은 형태** 가 된다. $T=0$ 에서 $\beta\to\infty$ — 시간 방향이 무한히 길어져 진짜 추가 차원이 된다.

핵심은 시간과 공간이 다르게 스케일된다는 점이다. 상관길이 $\xi$ 와 상관시간 $\xi_\tau$ 의 관계가

$$\xi_\tau \sim \xi^{\,z}$$

로, 동적 임계지수 $z$ 가 둘의 비등방성을 잰다. 따라서 유효 고전 차원은 $d_{\text{eff}} = d + z$ 다. $\square$

TFIM($d=1$, $z=1$)은 $1+1 = 2$ 차원 고전 이징 모형과 같은 보편성류 — 정확히 일치한다(2D 이징의 $\nu=1$ 과 TFIM의 $\nu=1$ 이 같다). 이것이 양자–고전 대응의 직접 증거다.

이 대응이 강력한 이유: 고전 임계현상의 방대한 지식(재규격화군, 보편성류, 임계지수)을 $T=0$ 양자 전이로 그대로 가져올 수 있다. 단, $z\neq 1$ 이면 시간과 공간이 다르게 스케일되어 비등방 임계현상이 되고, $z$ 자체가 새로운 보편 지수가 된다.

### 5. 상부 임계차원과 평균장

고전 이징의 상부 임계차원은 $d_c=4$ 다. 양자 전이의 유효 차원은 $d+z$ 이므로, $d+z \geq 4$ 이면 평균장 지수가 정확해진다. 예컨대 $z=2$ 인 어떤 양자 전이는 $d=2$ 에서 이미 $d+z=4=d_c$ 가 되어 평균장이 (로그 보정과 함께) 성립한다. 즉 **양자성이 유효 차원을 끌어올려 요동을 약화** 시킬 수 있다 — 고전계에서는 더 높은 차원에 가야 만나는 평균장 거동을, 양자계는 낮은 공간 차원에서도 만난다.

### 실험 1 — 1D 횡자기장 이징의 갭 닫힘과 자화

정확 대각화로 유한 사슬의 갭 $\Delta(g)$ 과 질서변수 $\langle|\sigma^z|\rangle$ 가 $g_c=1$ 부근에서 임계 거동을 보이는지 확인한다.

```python
import numpy as np
from numpy import kron
from numpy.linalg import eigh

# 파울리 행렬
sx = np.array([[0, 1], [1, 0]], float)
sz = np.array([[1, 0], [0, -1]], float)
I2 = np.eye(2)

def op_at(op, i, N):
    """N개 사이트 중 i번째에 op를, 나머지에 항등을 텐서곱"""
    M = np.array([[1.0]])
    for j in range(N):
        M = kron(M, op if j == i else I2)
    return M

def H_TFIM(N, J, h, pbc=True):
    """H = -J Σ σ^z_i σ^z_{i+1} - h Σ σ^x_i"""
    dim = 2**N
    H = np.zeros((dim, dim))
    for i in range(N):
        j = (i + 1) % N
        if j == 0 and not pbc:
            continue
        H -= J * op_at(sz, i, N) @ op_at(sz, j, N)
    for i in range(N):
        H -= h * op_at(sx, i, N)
    return H

N, J = 10, 1.0
gs = np.linspace(0.0, 2.0, 41)        # g = h/J 를 0→2 로 스캔
gaps, mags = [], []
Mz = sum(op_at(sz, i, N) for i in range(N)) / N   # 평균 자화 연산자

for g in gs:
    H = H_TFIM(N, J, J * g, pbc=True)
    E, V = eigh(H)
    gaps.append(E[1] - E[0])                       # 바닥-첫들뜸 갭
    psi = V[:, 0]
    mags.append(np.sqrt(psi @ (Mz @ Mz) @ psi))    # √<Mz²> (부호 축퇴 회피)

gaps, mags = np.array(gaps), np.array(mags)
imin = np.argmin(gaps)
print(f"N={N}  유한계 최소 갭: g={gs[imin]:.3f}, Δ={gaps[imin]:.4f}")
print(" g     Δ(gap)   √<Mz²>")
for k in range(0, len(gs), 5):
    print(f"{gs[k]:.2f}   {gaps[k]:.4f}   {mags[k]:.4f}")
# 기대: 유한계에서 갭 최소가 g≈1 부근, N↑일수록 g_c=1 로 수렴.
# √<Mz²> 은 g<1 에서 크고 g>1 에서 작아짐 — 강자성→상자성.
# 정확해 Δ_∞ = 2J|1-g| (열역학 극한)와 대조하면 유한크기 효과가 보인다.
```

이 코드를 실행하면 갭의 최소가 $g\approx 1$ 부근에 있고($N$ 을 키우면 정확히 $g_c=1$ 로 수렴), 자화가 $g_c$ 를 지나며 떨어지는 것을 본다. 무한계 정확해 $\Delta=2J|1-g|$ 와 비교하면 유한크기 효과(갭이 완전히 닫히지 않음)도 정량적으로 드러난다.

## 🧪 실험·관측 증거

- **$\mathrm{CoNb_2O_6}$ (Coldea et al., *Science* 2010)**: 준1차원 이징 강자성체에 횡자기장을 걸어 양자 임계점을 통과시키고, 임계점 근처의 들뜸 스펙트럼에서 예측된 $E_8$ 질량비($m_2/m_1 = 1.618\ldots$, 황금비)를 중성자·테라헤르츠 분광으로 관측 — TFIM 양자 임계성의 정밀 검증.
- **냉원자 초유체–모트 전이 (Greiner et al., *Nature* 2002)**: 광격자 깊이를 제어변수로 $T\approx 0$ 전이를 직접 관측, 간섭 패턴의 소멸로 위상 결맞음 상실을 확인.
- **$\mathrm{YbRh_2Si_2}$**: 자기장으로 반강자성 양자 임계점을 통과, 비열 $C/T \sim \ln(1/T)$ 의 발산으로 임계점의 그림자를 유한 온도에서 측정.

## 🔬 경계

- **정확히 풀리는 모형은 드물다.** TFIM은 Jordan–Wigner로 자유 페르미온이 되어 풀리지만, 2차원 이상 또는 상호작용이 더해지면 해석해가 없다. 정확 대각화는 $\sim 20$ 사이트, DMRG는 1차원에 한정된다(다음 문서들과 Ch6-05).
- **$z$ 가 정수가 아니거나 모형 의존적** 일 수 있다. 특히 금속의 양자 임계성에서는 페르미면이 동역학에 개입해 단순한 양자–고전 대응이 깨진다(Hertz–Millis 이론의 한계, 다음 문서).
- **유한 온도** 에서는 양자 임계점이 한 점이 아니라 부채꼴로 번지며, 양자와 열 효과가 얽힌다 — 깨끗한 $T=0$ 거동을 직접 보기 어렵다.

## 🧬 횡단 원리

- **자발 대칭 깨짐(SSB)** — TFIM의 $\mathbb{Z}_2$ 대칭이 $g<g_c$ 에서 깨지는 것은 Ch1-04 의 질서변수 언어 그대로다. 양자 전이는 SSB가 *온도가 아닌 양자 요동* 으로 일어나는 판이다(→ symmetry-conservation, → L2 phase-transitions).
- **재규격화군** — 양자–고전 대응으로 고전 임계현상의 RG가 그대로 이식된다. 보편성류는 차원과 대칭만으로 정해진다는 통찰이 $d\to d+z$ 로 확장된다(→ L2 phase-transitions).
- **얽힘** — 양자 임계점에서 얽힘 엔트로피가 $S\sim \frac{c}{6}\ln L$ 로 로그 발산하고 중심전하 $c$ 가 보편성류를 표지한다 — Ch5-04 의 얽힘이 상을 규정한다는 주제가 임계점에서 정점에 이른다(→ quantum-information).

## 🪜 창발

위 레이어에서 양자–고전 대응은 더 극적인 형태로 회귀한다. **L5 holographic-principle** 에서 강상관 양자 임계계는 한 차원 높은 중력 이론(AdS/CFT)으로 기술되며, 동적 임계지수 $z$ 가 시공간 계량의 비등방 스케일링(Lifshitz 기하)으로 나타난다. **L5 emergent-spacetime** 에서는 허수시간 차원이 진짜 창발 차원이 되어, "시간이 얽힘에서 나온다" 는 아이디어로 이어진다. 검증: 홀로그래피가 예측하는 임계 수송계수(예: 보편 전단점성 $\eta/s \geq \hbar/4\pi k_B$)를 강상관 양자 임계계에서 측정하는 것.

## 📐 예측·반증

- **갭 닫힘**: TFIM에서 $\Delta = 2J|1-g|$, $g_c=1$ 에서 정확히 0. 다른 $g_c$ 나 비선형 닫힘이 관측되면 모형이 틀렸다.
- **임계 분산**: 임계점에서 $\varepsilon_k \sim |k|^z$ 로 $z=1$(선형). 중성자 산란에서 측정 가능.
- **보편성류**: $d=1$ TFIM은 $2$D 고전 이징과 같아 $\nu=1$, $\beta=1/8$, $c=1/2$. 임계지수가 이 값에서 벗어나면 양자–고전 대응이나 보편성류 가정이 깨진 것.
- **유효 차원**: $d+z \geq 4$ 이면 평균장 지수. 이를 위반하는 비평균장 거동이 고차원에서 나오면 반증.

## 🤔 다음 질문

$T=0$ 양자 임계점은 닿을 수 없는 한 점이다. 그런데 이 점이 유한 온도 위로 *그림자* 를 드리운다면 — 그 그림자 안의 금속은 어떻게 행동하는가? 준입자가 사라지고 저항이 온도에 정비례하는 이상 금속(스트레인지 메탈)이 바로 그곳에서 나온다. 다음 문서에서 양자 임계 부채꼴과 비페르미 액체를 유도한다.

---

🧩 **Principle** — 절대영도에서 결합상수 $g$ 의 양자 요동이 바닥상태 에너지를 $g_c$ 에서 비해석적으로 만들고, $d$ 차원 양자 전이는 $d+z$ 차원 고전 전이로 사상되어 동적 임계지수 $z$ 가 새 차원의 비등방을 잰다.
🔬 **Boundary** — 정확해는 1차원·무상호작용에 국한되고, 금속의 페르미면이 개입하면 단순 양자–고전 대응이 깨진다.
🪜 **Emergence** — 위 레이어에서 허수시간 차원이 홀로그래피의 창발 공간 차원이 되며, 보편 수송계수($\eta/s$ 한계)로 검증된다.

## 📝 연습문제

**1. (기초)** TFIM의 자유 페르미온 분산 $\varepsilon_k = 2J\sqrt{1+g^2-2g\cos k}$ 에서 갭이 닫히는 $g$ 와 $k$ 를 구하라.

<details><summary>해설</summary>

갭은 $\min_k \varepsilon_k$. $\varepsilon_k \geq 0$ 이므로 근호 안 $f(k)=1+g^2-2g\cos k$ 를 최소화한다. $\cos k$ 가 최대($=1$, $k=0$)일 때 $f=(1-g)^2$, 최소($=-1$, $k=\pi$)일 때 $f=(1+g)^2$. $g>0$ 이면 $k=0$ 에서 $f=(1-g)^2$ 가 작다. 따라서 $\Delta = 2J|1-g|$, $g=1$ 에서 $\Delta=0$. 임계점에서 $k\to0$ 전개하면 $\varepsilon_k \approx 2J\sqrt{(1-g)^2 + gk^2}$, $g=1$ 대입 시 $\varepsilon_k\approx 2J|k|$ → 선형 분산, $z=1$. $\square$
</details>

**2. (심화)** 양자–고전 대응에서 $d=1$ TFIM이 $d=2$ 고전 이징과 같음을 분배함수로 보여라. 힌트: Trotter 분해 $e^{-\beta H} \approx (e^{-\Delta\tau H_0} e^{-\Delta\tau H_1})^M$, $M=\beta/\Delta\tau$.

<details><summary>해설</summary>

$H=H_0+H_1$, $H_0=-J\sum\sigma^z\sigma^z$(대각), $H_1=-h\sum\sigma^x$(비대각). Trotter로 시간을 $M$ 슬라이스로 쪼개면 각 슬라이스에서 $\sigma^z$ 기저 합을 삽입한다. $e^{-\Delta\tau H_0}$ 는 같은 시간 슬라이스 내 공간 이웃 스핀의 이징 결합을 준다. $e^{\Delta\tau h \sigma^x}$ 는 $\langle s_\tau | e^{\Delta\tau h\sigma^x}|s_{\tau+1}\rangle \propto e^{K_\tau s_\tau s_{\tau+1}}$ 형태로 **시간 방향 이징 결합** $K_\tau = -\frac12\ln\tanh(\Delta\tau h)$ 를 만든다. 결과는 공간(원래 사슬)+시간(Trotter 방향) 2차원 격자 위의 이방성 고전 이징 분배함수. 연속극한에서 등방 2D 이징과 같은 보편성류 → $\nu=1$ 일치. $z=1$ 이라 공간·시간이 같은 거듭제곱으로 스케일됨이 핵심. $\square$
</details>

**3. (논문 비평)** Coldea et al. (*Science* 2010)은 $\mathrm{CoNb_2O_6}$ 에서 황금비 $E_8$ 질량비를 관측했다고 주장한다. 이 주장이 "TFIM 양자 임계성의 증거" 가 되려면 어떤 조건들이 충족되어야 하며, 어디서 의심할 수 있는가?

<details><summary>해설</summary>

$E_8$ 질량 스펙트럼은 임계 2D 이징(=1D TFIM 양자 임계점)에 *세로* 자기장 섭동을 더했을 때 나타나는 적분가능 장론의 예측이다. 증거가 되려면: (1) 물질이 충분히 이징적이어야 함 — 실제 $\mathrm{CoNb_2O_6}$ 는 준1차원이고 사슬 간 결합·DM 상호작용이 있어 순수 TFIM에서 벗어남. (2) 횡자기장이 정확히 임계점에 맞춰져야 하나 실험은 임계점 *근처*. (3) 관측된 비 $m_2/m_1\approx1.6$ 가 황금비 $1.618$ 과 일치한다지만 측정 불확도와 사슬간 결합 보정 안에서의 일치인지 검토 필요. 비판점: 단일 비율의 일치는 강력하지만 $E_8$ 의 8개 질량 전부를 보지 못했고, 고차 질량은 사슬간 결합·연속체에 묻힌다. 즉 "정성적으로 인상적, 정량적으로 부분적" 인 증거다. $\square$
</details>
