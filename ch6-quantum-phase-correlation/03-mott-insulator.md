# 모트 절연체 — 상호작용이 만드는 절연체

> *밴드 이론은 단언한다 — 밴드가 반쯤 차면 금속이다. 그러나 어떤 물질은 반쯤 채워도 절연체다. 상호작용 $U$ 가 충분히 크면, 전자가 서로를 피하느라 움직이지 못한다. 밴드가 아니라 상호작용이 절연체를 만든다.*

## 🎯 핵심 질문

- 밴드 이론은 "반채움 = 금속" 을 예측하는데, 왜 $\mathrm{NiO}$·$\mathrm{La_2CuO_4}$ 같은 물질은 절연체인가? 무엇이 밴드 이론을 위반하는가?
- 허바드 모형 $H=-t\sum c^\dagger c + U\sum n_\uparrow n_\downarrow$ 에서 $U\gg t$ 일 때 절연체 갭(모트–허바드 갭)이 어떻게 열리는가?
- 모트 절연체의 국소 스핀들은 왜 반강자성으로 정렬하는가? 초교환 $J=4t^2/U$ 는 어디서 오는가?

## 🌍 어디서 나타나나

- **전이금속 산화물**: $\mathrm{NiO}$, $\mathrm{CoO}$, $\mathrm{MnO}$, $\mathrm{V_2O_3}$ — 부분 채워진 $d$ 밴드를 가져 밴드 이론상 금속이어야 하나 절연체. 모트 절연체의 원형.
- **구리산화물 모체** $\mathrm{La_2CuO_4}$: 반채움 모트 절연체이자 반강자성체. 여기에 구멍을 도핑하면 고온 초전도가 나온다(다음 문서) — "도핑된 모트 절연체".
- **유기 전하이동 염** $\kappa\text{-(ET)}_2X$: 압력으로 모트 전이를 조율, 모트 절연체–금속 전이를 직접 관측.
- **냉원자 페르미 광격자**: $^6\mathrm{Li}$·$^{40}\mathrm{K}$ 로 허바드 모형을 양자 시뮬레이션, 모트 절연체와 반강자성 상관을 측정.
- **스핀 액체 후보** $\kappa\text{-(ET)}_2\mathrm{Cu_2(CN)_3}$: 모트 절연체이지만 반강자성으로 정렬하지 않고 스핀 액체로 남는 — 모트 절연체가 스핀 액체의 모체(Ch5)임을 보여주는 핵심.

## 🔍 직관의 함정

**함정 1 — "반쯤 찬 밴드는 무조건 금속이다."** 밴드 이론(단일전자 근사)의 결론이다. 페르미면이 밴드 안에 있으면 빈 상태가 바로 위에 있어 전류가 흐른다. 그러나 이 논리는 전자들이 *서로를 무시* 한다고 가정한다. 강한 쿨롱 반발이 있으면 이 가정이 무너진다.

**함정 2 — "절연체 = 가득 찬 밴드(band insulator)."** 밴드 절연체는 밴드가 완전히 차서 갭 위에만 빈 상태가 있는 경우다. 모트 절연체는 *밴드가 반밖에 안 찼는데도* 절연체다. 갭의 기원이 전혀 다르다 — 밴드 절연체는 결정 퍼텐셜, 모트 절연체는 전자–전자 상호작용 $U$.

**함정 3 — "$U$ 만 크면 항상 절연체다."** 비율 $U/t$ 가 문제다. $U\gg t$ 면 절연체, $U\ll t$ 면 금속. 그 사이 어딘가에 모트 전이가 있다. 또한 채움(filling)도 중요 — 반채움에서만 깨끗한 모트 절연체가 되고, 도핑하면 금속·초전도로 바뀐다.

## ⚙️ 제1원리 유도

### 1. 허바드 모형: 운동 vs 반발의 경쟁

가장 단순하면서 강상관의 정수를 담은 모형이 허바드 모형이다:

$$H = -t\sum_{\langle ij\rangle, \sigma} \left(c_{i\sigma}^\dagger c_{j\sigma} + \text{h.c.}\right) + U\sum_i n_{i\uparrow} n_{i\downarrow}.$$

- $-t$ 항: 이웃 사이트로의 호핑 — 전자를 *퍼뜨려* 운동에너지를 낮춘다(밴드를 만든다, 폭 $\sim 2zt$, $z$=배위수).
- $+U$ 항: 같은 사이트에 ↑↓ 둘이 있으면 에너지 $U$ 손해 — 전자를 *국소화* 시킨다(이중점유 회피).

두 항은 비가환($[H_t, H_U]\neq0$)이고 정반대를 원한다. 승부는 $U/t$ 가 가른다.

### 2. 두 극한: 원자 극한과 밴드 극한

**$t=0$ (원자 극한):** 사이트들이 끊어진다. 반채움(사이트당 전자 1개)에서 각 사이트는 ↑ 또는 ↓ 단일점유. 한 사이트에서 전자를 빼면 빈 사이트(에너지 0), 더하면 이중점유(에너지 $U$). 따라서 전하 들뜸 갭이 정확히 $U$ — 절연체다. 갭 $=U$. $\square$

**$U=0$ (밴드 극한):** 자유 긴밀결합. 분산 $\varepsilon_k = -2t\sum_\mu\cos k_\mu$, 반채움이면 페르미면이 밴드 중앙 — 금속.

핵심 통찰: $U=0$ 에서 금속, $U=\infty$ 에서 절연체. 둘은 질적으로 다르므로 **사이에 모트 전이** $U_c \sim$ 밴드폭 $W$ 가 있어야 한다. 밴드 이론은 $U=0$ 에서 출발하므로 큰 $U$ 의 절연체를 절대 보지 못한다.

### 3. 상하 허바드 밴드와 모트–허바드 갭

$U$ 가 크지만 유한할 때, 호핑이 원자 준위를 퍼뜨린다. 전하 들뜸의 두 종류를 보자. 반채움 바탕(모든 사이트 단일점유)에서:

- **전자 제거**(홀 생성): 빈 사이트가 생겨 이웃으로 호핑 → 폭 $\sim W$ 의 밴드. 이것이 **하부 허바드 밴드**(lower Hubbard band, LHB), 점유 상태.
- **전자 추가**(이중점유 생성): 추가 전자가 이중점유를 만들며 호핑 → 폭 $\sim W$ 의 밴드, 에너지 $\sim U$ 위. 이것이 **상부 허바드 밴드**(upper Hubbard band, UHB), 빈 상태.

두 밴드의 중심이 $U$ 만큼 떨어지고 각각 폭 $W$ 이므로, 모트–허바드 갭은

$$\Delta_{\text{Mott}} \approx U - W.$$

$U > W$ 이면 갭이 열린 절연체, $U < W$ 이면 두 밴드가 겹쳐 금속. 모트 전이는 $U_c\sim W$ 에서. $\square$ 이것이 밴드 절연체와의 결정적 차이 — 상하 허바드 밴드는 *상호작용이 쪼갠* 밴드이지, 결정 퍼텐셜이 만든 밴드가 아니다.

### 4. 강결합 전개: 초교환 $J=4t^2/U$

$U\gg t$ 의 모트 절연체에서 전하는 얼었지만 스핀은 살아 있다. 각 사이트의 스핀이 어떻게 상호작용하는가? 2차 섭동론으로 유도한다.

반채움 두 사이트를 보자. 바탕 다양체는 이중점유 없는 상태들 — ↑↑, ↑↓, ↓↑, ↓↓ (에너지 0). 호핑 $t$ 는 이들을 이중점유 중간상태(에너지 $U$)를 거쳐 섞는다.

평행 스핀(↑↑): 파울리 배타로 호핑 불가 — 같은 스핀이 같은 사이트에 못 옴. 에너지 보정 0.

반평행 스핀(↑↓): ↑ 이 오른쪽으로 호핑하면 오른쪽 사이트가 (↑↓) 이중점유(에너지 $+U$), 다시 호핑해 돌아오거나 ↓ 가 왼쪽으로. 2차 섭동:

$$\Delta E = -\sum_{\text{중간상태}} \frac{|\langle \text{중간}|H_t|\text{초기}\rangle|^2}{E_{\text{중간}} - E_{\text{초기}}} = -\frac{t^2}{U} - \frac{t^2}{U} = -\frac{2t^2}{U}.$$

(두 경로: ↑ 가 오른쪽으로, 또는 ↓ 가 왼쪽으로 — 각각 $t^2/U$.) 반평행 스핀이 에너지를 낮춘다 → **반강자성** 선호. 이 에너지 차이를 하이젠베르크 형태로 쓰면

$$H_{\text{eff}} = J\sum_{\langle ij\rangle}\mathbf{S}_i\cdot\mathbf{S}_j, \qquad J = \frac{4t^2}{U} > 0,$$

반강자성 교환상수다. $\square$ ($\mathbf{S}_i\cdot\mathbf{S}_j$ 의 평행/반평행 고윳값 차이가 위 $2t^2/U$ 를 재현하도록 계수가 $4t^2/U$ 가 된다.) 이것이 **초교환**(superexchange) — 직접 자기 상호작용이 아니라 *가상 호핑* 이 매개하는 반강자성. 모트 절연체가 반강자성체인 미시 기원이다.

핵심 함의: 모트 절연체는 단순한 절연체가 아니라 **스핀 자유도가 살아 있는** 절연체다. 이 스핀들이 반강자성으로 정렬하거나(대부분), 기하학적 좌절로 정렬을 거부하고 **스핀 액체**(Ch5)가 되거나, 구멍을 도핑하면 고온 초전도(다음 문서)가 된다.

### 5. 2-사이트 허바드의 정확해

2-사이트 반채움 허바드(전자 2개, $S_z=0$)는 손으로 풀린다. 기저는 4개: $|\!\uparrow,\downarrow\rangle$, $|\!\downarrow,\uparrow\rangle$, $|\!\uparrow\downarrow,0\rangle$, $|0,\uparrow\downarrow\rangle$. 호핑이 단일점유 상태와 이중점유 상태를 섞는다. 싱글렛 바닥상태 에너지는

$$E_0 = \frac{U - \sqrt{U^2 + 16t^2}}{2}.$$

트리플렛은 호핑이 막혀(파울리) $E_{\text{triplet}}=0$. 싱글렛–트리플렛 갭(=유효 교환):

$$J_{\text{exact}} = E_{\text{triplet}} - E_0 = \frac{\sqrt{U^2+16t^2} - U}{2} \xrightarrow{U\gg t} \frac{4t^2}{U}.$$

대형 $U$ 극한에서 정확히 초교환 $4t^2/U$ 로 수렴한다 — 2차 섭동론의 검증. $\square$

### 실험 1 — 2-사이트 허바드: 모트 갭과 초교환 vs U

2-사이트 허바드를 정확 대각화로 풀어 (i) 전하 갭이 $U$ 와 함께 열리고, (ii) 스핀 교환 $J$ 가 $4t^2/U$ 로 수렴함을 확인한다.

```python
import numpy as np
from itertools import combinations

# 2-사이트 허바드 정확 대각화 (스핀-궤도 4개: 0=1↑,1=1↓,2=2↑,3=2↓)
norb = 4

def make_basis(Ne, Sz):
    """전자수 Ne, 스핀 Sz 섹터의 Fock 기저"""
    up_orb, dn_orb = {0, 2}, {1, 3}
    basis = []
    for occ in combinations(range(norb), Ne):
        nup = sum(o in up_orb for o in occ)
        ndn = sum(o in dn_orb for o in occ)
        if nup - ndn == Sz:
            basis.append(occ)
    return basis

def cdag_c(occ, p, q):
    """c_p† c_q 를 Fock 상태 occ 에 작용 (페르미온 부호 포함)"""
    occ = set(occ)
    if q not in occ:
        return None
    s1 = (-1) ** sum(o < q for o in occ); occ.remove(q)
    if p in occ:
        return None
    s2 = (-1) ** sum(o < p for o in occ); occ.add(p)
    return tuple(sorted(occ)), s1 * s2

def hamiltonian(basis, t, U):
    idx = {s: i for i, s in enumerate(basis)}
    n = len(basis)
    H = np.zeros((n, n))
    hops = [(0, 2), (1, 3)]            # 사이트1↑-사이트2↑, 사이트1↓-사이트2↓
    for ket in basis:
        b = idx[ket]
        d = (0 in ket and 1 in ket) + (2 in ket and 3 in ket)  # 이중점유 사이트 수
        H[b, b] += U * d
        for a, c in hops:
            for p, q in [(a, c), (c, a)]:
                res = cdag_c(ket, p, q)
                if res and res[0] in idx:
                    H[idx[res[0]], b] += -t * res[1]
    return H

t = 1.0
print(" U     E0(싱글렛)   J_exact   4t²/U")
for U in [1, 2, 4, 8, 16, 32]:
    b0 = make_basis(2, 0)             # 반채움, Sz=0
    E_sz0 = np.linalg.eigvalsh(hamiltonian(b0, t, U))[0]
    # 트리플렛: Sz=+1 섹터의 바닥 (호핑 막혀 E=0)
    b1 = make_basis(2, 2)
    E_sz1 = np.linalg.eigvalsh(hamiltonian(b1, t, U))[0]
    J = E_sz1 - E_sz0
    print(f"{U:4d}   {E_sz0:8.4f}   {J:7.4f}   {4*t**2/U:6.4f}")

# 전하 갭: E(N=3)+E(N=1)-2E(N=2) (반채움 N=2 기준 입자/홀 추가)
print("\n전하 갭 Δc = E(3)+E(1)-2E(2):")
for U in [1, 2, 4, 8, 16]:
    E2 = np.linalg.eigvalsh(hamiltonian(make_basis(2, 0), t, U))[0]
    E1 = np.linalg.eigvalsh(hamiltonian(make_basis(1, 1), t, U))[0]
    E3 = np.linalg.eigvalsh(hamiltonian(make_basis(3, 1), t, U))[0]
    print(f"  U={U:3d}  Δc = {E3 + E1 - 2*E2:.4f}")
# 기대: J → 4t²/U (대형 U), 전하 갭이 U와 함께 커짐 → 모트 절연체.
```

이 코드는 2-사이트 허바드를 정확 대각화한다. 출력에서 $J_{\text{exact}}=(\sqrt{U^2+16}-U)/2$ 가 대형 $U$ 에서 $4t^2/U$ 로 수렴하고(예: $U=32$ 에서 둘 다 $\approx0.125$), 전하 갭이 $U$ 와 함께 커지는 — 모트 절연체의 형성을 본다.

## 🧪 실험·관측 증거

- **$\mathrm{NiO}$ 의 광학 갭 $\sim 4$ eV**: 밴드 이론(LDA)은 금속을 예측하나 실제는 큰 절연체 갭 — 모트(혹은 전하이동) 절연체의 고전적 증거. $U\sim 8$ eV가 필요.
- **$\mathrm{La_2CuO_4}$ 의 반강자성**: 모트 절연체 모체가 네일 온도 $T_N\approx 320$ K의 반강자성체. 측정된 교환 $J\approx 130$ meV가 $4t^2/U$ ($t\approx0.4$ eV, $U\approx4$ eV)와 일치.
- **냉원자 허바드 양자 시뮬레이션 (Mazurenko et al., *Nature* 2017)**: $^6\mathrm{Li}$ 광격자에서 반채움 모트 절연체의 반강자성 스핀 상관을 단일 사이트 분해능으로 직접 관측.
- **$\mathrm{V_2O_3}$ 의 모트 전이**: 압력·온도로 절연체–금속 전이를 직접 조율, 1차 전이의 이력곡선 관측.

## 🔬 경계

- **2-사이트는 모트 전이를 못 본다**: 위 정확해는 항상 갭이 있다(유한계). 진짜 모트 *전이*($U_c$ 에서 금속↔절연체)는 열역학 극한이 필요하고, 동적 평균장 이론(DMFT)이 표준 도구다(Ch6-05).
- **전하이동 절연체와의 구별**: $\mathrm{NiO}$·구리산화물은 순수 모트가 아니라 산소 $p$ 밴드가 끼어든 전하이동 절연체(Zaanen–Sawatzky–Allen 분류). 갭이 $U$ 가 아니라 전하이동 에너지 $\Delta_{pd}$ 로 결정.
- **도핑된 모트 절연체는 미해결**: 반채움은 깨끗하나 구멍을 넣으면(다음 문서) 스트레인지 메탈·의사갭·고온 초전도가 나오고, 이를 통제된 이론으로 풀지 못한다.

## 🧬 횡단 원리

- **상호작용이 만드는 질서(Ch1-04, Ch2-05)** — 평균장(밴드 이론)이 무너지고 상관이 질적으로 새 상(절연체)을 만드는 곳. 평균장 너머의 대표 사례(→ Ch2-05).
- **창발하는 스핀 모형** — 전하를 적분해내면 허바드가 하이젠베르크 $J\mathbf{S}\cdot\mathbf{S}$ 로 환원된다. 미시(전자+쿨롱)에서 유효(스핀)로의 창발 — 유효이론의 정수(→ L6 emergence).
- **스핀 액체의 모체(Ch5)** — 좌절된 모트 절연체는 반강자성 대신 긴 거리 얽힘의 스핀 액체가 된다. 모트가 위상 질서·창발 게이지장의 출발점(→ Ch5-01·03).

## 🪜 창발

모트 절연체에 구멍을 도핑하면 강상관 금속이 되고, 이 도핑된 모트 절연체가 **L5 holographic-principle** 의 강상관 홀로그래피가 겨냥하는 바로 그 계다 — 준입자 없는 금속을 중력 쌍대로 기술. **L6 emergence** 에서 허바드→하이젠베르크 환원은 "전하 자유도가 얼면 스핀 유효이론이 창발한다" 는 유효이론 계층의 교과서적 예다. 검증: 도핑된 모트 절연체의 상도(반강자성→의사갭→스트레인지 메탈→초전도)를 허바드 모형의 수치(DMFT·텐서망)나 냉원자 시뮬레이터로 재현하는 것.

## 📐 예측·반증

- **갭의 채움 의존성**: 반채움에서만 모트 절연체, 도핑하면 금속. 임의 채움에서 절연체가 유지되면 모트가 아님.
- **초교환 스케일링**: $J = 4t^2/U$ — 압력으로 $t$ 를 바꾸면 $J\propto t^2$, $U$ 를 바꾸면 $J\propto 1/U$. 중성자로 측정한 마그논 분산에서 검증.
- **모트–허바드 갭**: $\Delta\approx U-W$. 광학·광전자분광에서 상하 허바드 밴드를 분리해 관측. 갭이 $W$ 대신 $U$ 로 스케일하면 모트 확정.
- **모트 전이 임계비**: $U_c \sim W$(배위수·차원 의존). DMFT는 무한차원에서 $U_c\approx 1.5W$ 류 예측 — 위반 시 평균장 가정 붕괴.

## 🤔 다음 질문

반채움 모트 절연체는 절연체이자 반강자성체다. 그런데 여기에 구멍을 몇 % 도핑하면 — 반강자성이 무너지고, 의사갭이 열리고, 스트레인지 메탈이 되고, 어떤 도핑에서 *고온 초전도* 가 나온다. 이 "도핑된 모트 절연체" 가 왜 그렇게 풍부하고 왜 아직 메커니즘이 미해결인가? 다음 문서에서 고온 초전도를 정직하게 다룬다.

---

🧩 **Principle** — 반채움에서 강한 상호작용 $U\gg t$ 가 밴드를 상하 허바드 밴드로 쪼개 갭 $\Delta\approx U-W$ 의 절연체를 만들고(밴드 이론 위반), 남은 스핀이 가상 호핑으로 매개된 초교환 $J=4t^2/U$ 의 반강자성으로 정렬한다.
🔬 **Boundary** — 모트 *전이* 와 도핑된 모트 절연체는 정확 대각화·섭동론을 넘어서며 DMFT·텐서망조차 통제하기 어렵다.
🪜 **Emergence** — 위 레이어에서 도핑된 모트 절연체는 강상관 홀로그래피의 표적이고, 허바드→하이젠베르크 환원은 유효이론 창발의 모범이며, 상도 재현으로 검증된다.

## 📝 연습문제

**1. (기초)** 원자 극한($t=0$) 반채움에서 전하 갭이 정확히 $U$ 임을 보여라.

<details><summary>해설</summary>

$t=0$ 이면 사이트가 독립. 반채움 = 사이트당 전자 1개(단일점유), 에너지 0(이중점유 없음). 한 사이트에서 전자를 빼면 빈 사이트 → 에너지 변화 0(그 사이트가 이중점유였던 적 없음). 전자를 더하면 어떤 사이트가 이중점유 → 에너지 $+U$. 전하 갭 = (전자 추가 에너지) − (전자 제거 에너지) = $U - 0 = U$. 광전자분광에서 LHB는 $E_F$ 아래, UHB는 $E_F$ 위 $U$ 만큼 — 정확히 갭 $U$. $\square$
</details>

**2. (심화)** 2-사이트 허바드 싱글렛 바닥상태 에너지 $E_0=(U-\sqrt{U^2+16t^2})/2$ 를 직접 대각화로 유도하라. 힌트: $S_z=0$ 섹터의 4×4를 대칭화하라.

<details><summary>해설</summary>

$S_z=0$ 기저: 단일점유 $|\!\uparrow\downarrow\rangle\equiv|\!\uparrow,\downarrow\rangle$, $|\!\downarrow\uparrow\rangle$ 와 이중점유 $|2,0\rangle=|\!\uparrow\downarrow,0\rangle$, $|0,2\rangle$. 호핑은 단일점유↔이중점유를 섞는다. 대칭/반대칭 조합으로: 트리플렛 $|t\rangle=(|\!\uparrow\downarrow\rangle-|\!\downarrow\uparrow\rangle)/\sqrt2$ 는 호핑이 0(파울리) → $E=0$. 싱글렛 부분공간은 $|s\rangle=(|\!\uparrow\downarrow\rangle+|\!\downarrow\uparrow\rangle)/\sqrt2$ 와 이중점유 대칭조합 $|d\rangle=(|2,0\rangle+|0,2\rangle)/\sqrt2$. 이 2×2: $\langle s|H|s\rangle=0$, $\langle d|H|d\rangle=U$, $\langle s|H|d\rangle=-2t$(부호·인자 주의). 행렬 $\begin{pmatrix}0 & -2t\\ -2t & U\end{pmatrix}$ 의 고윳값 $\lambda=(U\mp\sqrt{U^2+16t^2})/2$. 바닥 $E_0=(U-\sqrt{U^2+16t^2})/2$. 대형 $U$: $E_0\approx -4t^2/U$, 트리플렛과의 갭 $=4t^2/U=J$. $\square$
</details>

**3. (논문 비평)** Mazurenko et al. (*Nature* 2017)은 냉원자 광격자에서 반채움 허바드의 반강자성 상관을 관측했다. 이 양자 시뮬레이션이 "고체 모트 절연체의 충실한 모형" 이라는 주장의 강점과 한계는?

<details><summary>해설</summary>

강점: (1) 깨끗한 단일밴드 허바드를 *정확히* 구현 — 고체에 있는 포논·다중밴드·무질서가 없어 이론과 직접 비교 가능. (2) 단일 사이트 분해능으로 스핀 상관을 *직접* 측정 — 고체에서는 평균량만 측정. (3) $U/t$ 를 광격자 깊이로 자유 조율. 한계: (1) 도달 온도가 $T\sim J$ 수준이라 진짜 저온 반강자성 *장거리 질서* 에는 못 미침 — 짧은 거리 상관만. (2) 시스템 크기·수명 제한으로 열역학 극한 외삽이 어려움. (3) 고체의 전하이동 물리(산소 $p$ 밴드)는 단일밴드 허바드에 없음 — 실제 구리산화물과 단순 허바드 사이엔 간극. 즉 "허바드 모형의 충실한 시뮬레이터" 이지 "구리산화물의 충실한 모형" 은 아직 아니다 — 모형과 물질을 구별해야 한다. $\square$
</details>
