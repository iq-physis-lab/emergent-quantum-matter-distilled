# 강상관의 최전선 — 풀리지 않은 다체

> *고온 초전도가 미해결인 이유는 자연이 비밀스러워서가 아니라, 우리가 강상관 다체 문제를 풀 도구가 없어서다. 부호 문제가 몬테카를로를 무력화하고, 얽힘 장벽이 텐서망을 막는다. 이 문서는 정직한 미해결의 해부학이다.*

## 🎯 핵심 질문

- 강상관 다체 문제는 *왜* 어려운가? 단지 계산이 큰 것인가, 아니면 *원리적* 장벽이 있는가?
- 양자 몬테카를로의 부호 문제(sign problem)는 정확히 무엇이고, 왜 NP-난해로 알려졌는가?
- DMRG·텐서망의 얽힘 장벽은 무엇이고, 왜 2차원·시간 발전에서 무너지는가?
- 그렇다면 최전선의 도구(DMFT, 양자 시뮬레이터)는 무엇을 우회하는가?

## 🌍 어디서 나타나나

- **고온 초전도 위상도(Ch6-04)**: 도핑된 허바드/$t$-$J$ 의 바닥상태·유한온도를 통제된 정확도로 풀지 못해 메커니즘이 미해결.
- **양자 자성·스핀 액체(Ch5)**: 좌절된 반강자성체(카고메·삼각격자)의 바닥상태가 정렬인지 스핀 액체인지를 결정하는 데 수치 장벽.
- **핵물질·중성자별**: 강상관 페르미온의 상태방정식을 격자 QCD가 유한 밀도에서 부호 문제로 못 풂.
- **양자 시뮬레이터**: 냉원자 광격자·초전도 큐비트·이온 트랩이 *고전적으로 풀 수 없는* 강상관 모형을 물리적으로 "풀어" 보려는 시도 — 부호 문제를 자연에 떠넘긴다.

## 🔍 직관의 함정

**함정 1 — "컴퓨터가 빨라지면 풀린다."** 아니다. 부호 문제는 원리적 장벽일 가능성이 높다(NP-난해 증명, Troyer–Wiese 2005). 무어의 법칙으로 해결될 종류가 아니다 — 비용이 시스템 크기·역온도에 *지수적* 으로 폭발한다.

**함정 2 — "정확 대각화가 있으니 작은 계는 정확히 안다."** 맞지만 힐베르트 공간이 $4^N$($N$ 사이트 허바드)으로 폭발해 $N\sim 20$ 이 한계다. $20$ 사이트로는 열역학 극한·임계 거동을 외삽하기 어렵다 — 특히 긴 상관길이를 가진 상에서.

**함정 3 — "DMRG가 1D를 풀었으니 2D도 곧 풀린다."** 1D의 성공은 면적 법칙(Ch5-04) 덕이다. 1D 바닥상태의 얽힘이 작아($S\sim$상수) 적은 변분 매개변수로 표현된다. 2D는 면적 법칙이 $S\sim L$ 로 커져 필요한 결합차원이 *지수적* 으로 늘어난다 — 이것이 얽힘 장벽이다.

## ⚙️ 제1원리 유도

### 1. 왜 다체가 지수적으로 어려운가

$N$ 개 스핀-1/2의 힐베르트 공간 차원은 $2^N$. 일반 양자상태는 $2^N$ 개 복소진폭을 갖는다 — $N=300$ 이면 우주의 원자 수보다 많다. 해밀토니안 대각화는 $O(2^{3N})$ 연산, 정확 대각화는 $N\lesssim20$ 에서 멈춘다.

핵심 질문: *물리적* 바닥상태는 이 거대 공간의 작은 구석(면적 법칙 얽힘, Ch5-04)에 산다. 그렇다면 효율적으로 표현·계산할 수 있어야 하지 않나? 답: *어떤* 상은 그렇고(1D 갭 있는 상 → DMRG), *어떤* 상은 그렇지 않다(2D 임계·부호 문제 계). 무엇이 가능/불가능을 가르는지가 이 문서의 주제다.

### 2. 부호 문제: 양자 몬테카를로의 벽

양자 몬테카를로(QMC)는 양자 분배함수를 고전 확률 합으로 사상해 표본추출한다. 허수시간 경로적분(Ch6-01):

$$Z = \mathrm{Tr}\,e^{-\beta H} = \sum_{\mathcal{C}} W(\mathcal{C}),$$

여기서 $\mathcal{C}$ 는 구성(configuration), $W(\mathcal{C})$ 는 그 가중치. **$W(\mathcal{C}) \geq 0$ 이면** $W$ 를 확률분포로 보고 몬테카를로 표본추출 — 통계오차가 $1/\sqrt{N_{\text{samples}}}$ 로 줄어든다. 다항시간에 풀린다.

문제는 페르미온이다. 페르미온 교환의 반대칭성(Ch2)이 가중치에 음수를 만든다: $W(\mathcal{C})$ 가 부호가 바뀐다. 그러면 $W$ 를 확률로 못 쓴다. 우회책은 $|W|$ 로 표본추출하고 부호를 관측량에 넣는 것:

$$\langle O\rangle = \frac{\sum_\mathcal{C} O(\mathcal{C}) \,\mathrm{sgn}(\mathcal{C})\,|W(\mathcal{C})|}{\sum_\mathcal{C}\mathrm{sgn}(\mathcal{C})\,|W(\mathcal{C})|} = \frac{\langle O\cdot\mathrm{sgn}\rangle_{|W|}}{\langle\mathrm{sgn}\rangle_{|W|}}.$$

여기서 평균 부호 $\langle\mathrm{sgn}\rangle$ 가 핵심이다. 양수·음수 가중치가 거의 상쇄되면 $\langle\mathrm{sgn}\rangle$ 가 0에 가까워진다. 통계역학적으로

$$\langle\mathrm{sgn}\rangle = \frac{Z}{Z_{|W|}} = e^{-\beta N \Delta f},$$

여기서 $\Delta f \geq 0$ 는 실제 계와 부호를 무시한 보존계의 자유에너지 밀도 차. 따라서 평균 부호가 시스템 크기 $N$ 과 역온도 $\beta$ 에 **지수적으로 작아진다**. $\square$

상대오차는 $1/(\langle\mathrm{sgn}\rangle\sqrt{N_{\text{samples}}})$ 이므로, 정해진 정밀도를 위한 표본 수가 $\propto e^{2\beta N \Delta f}$ — **지수적 폭발**. 이것이 부호 문제다. Troyer–Wiese(2005)는 일반 부호 문제를 다항시간에 푸는 것이 NP-난해임을 증명했다 — 즉 *일반적 해법은 거의 확실히 없다*. 도핑된 허바드(고온 초전도!)·좌절 자성이 바로 이 벽에 막힌다.

### 3. 얽힘 장벽: DMRG·텐서망의 벽

다른 접근은 변분 파동함수를 텐서망(tensor network)으로 압축하는 것이다. 1D의 행렬곱상태(MPS)는 바닥상태를

$$|\psi\rangle = \sum_{\{s\}} \mathrm{Tr}\left(A^{s_1}A^{s_2}\cdots A^{s_N}\right)|s_1\cdots s_N\rangle$$

로 쓴다. 각 $A^{s}$ 는 $\chi\times\chi$ 행렬, $\chi$ 가 결합차원(bond dimension). DMRG는 이를 변분 최적화한다.

핵심: MPS가 표현할 수 있는 얽힘은 $S \leq \ln\chi$ 로 제한된다. 즉 한 절단면을 가로지르는 얽힘 엔트로피가 $\ln\chi$ 를 넘으면 그 $\chi$ 로는 표현 불가. 1D 갭 있는 바닥상태는 면적 법칙으로 $S\sim$ 상수 → 작은 $\chi$ 로 충분 → DMRG 대성공. $\square$

2D는 다르다. $L\times L$ 격자를 1D로 풀어 쓰면(snake) 절단면을 가로지르는 얽힘이 $S\sim L$(2D 면적 법칙은 경계 길이 $\sim L$). 이를 표현하려면 $\chi \gtrsim e^{cL}$ — 시스템 폭에 *지수적* 결합차원. 메모리·연산이 $\chi^2$–$\chi^3$ 로 폭발한다. 이것이 얽힘 장벽이다. $\square$

시간 발전은 더 나쁘다. 비평형 동역학에서 얽힘이 시간에 *선형* 으로 증가($S\sim vt$, $v$=얽힘 속도)해 $\chi\sim e^{vt}$ — 짧은 시간만 닿는다. PEPS(2D 텐서망)·MERA가 부분적 우회를 주지만 계약(contraction)이 #P-난해라 또 다른 장벽.

### 4. 두 장벽의 통일된 그림

부호 문제와 얽힘 장벽은 *다른* 방법(QMC vs 텐서망)의 벽이지만, 둘 다 "강상관 다체가 고전적으로 어렵다" 는 같은 사실의 두 얼굴이다. 실제로 어떤 부호 문제는 기저 변환으로 완화되고(부호 문제는 표현 의존적), 어떤 얽힘은 영리한 텐서망으로 압축된다. 그러나 *일반적으로* 둘을 동시에 피할 수 없다 — 강하게 얽히고 페르미온적인 계가 강상관의 핵심이다. 양자 컴퓨터·시뮬레이터가 매력적인 이유: 양자계로 양자계를 모사하면 이 두 장벽을 *물리적으로* 우회한다(Feynman의 통찰).

### 5. 최전선의 도구들

완전 해법이 없으므로, 각자 다른 근사로 일부를 푼다:

- **동적 평균장 이론(DMFT)**: 격자를 *한 불순물 사이트 + 자기무모순 욕조* 로 사상한다. 무한차원에서 정확하고, 국소 상관(모트 전이!)을 잘 잡는다. 한계: 공간적 상관(비국소 짝짓기, $d$-파)을 놓침 — 군집 확장(CDMFT, DCA)이 부분 보완.
- **양자 몬테카를로 변종**: 행렬식 QMC(반채움 허바드는 부호 문제 없음!), 진단 몬테카를로, 제약 경로 QMC — 특정 모형·채움에서 부호를 피한다.
- **텐서망**: 1D는 DMRG로 거의 정확, 2D는 유한 폭 원통·PEPS로 부분 접근.
- **양자 시뮬레이터**: 냉원자 허바드(Ch6-03)·이온 트랩 스핀 모형이 고전적으로 못 푸는 영역을 물리적으로 탐색.

이 도구들의 *불일치* 가 곧 최전선의 지도다 — 같은 모형에 대해 방법마다 다른 답이 나오는 곳이 미해결의 현장이다.

### 실험 1 — 부호 문제의 지수적 폭발 시연

장난감 모형으로 평균 부호 $\langle\mathrm{sgn}\rangle$ 가 시스템 크기·역온도에 지수적으로 작아져 QMC 추정의 통계오차가 폭발함을 보인다.

```python
import numpy as np
rng = np.random.default_rng(0)

# 부호 문제 장난감: 가중치 W(C)가 부호를 가지며, <sgn> = exp(-β N Δf) 로 감소.
# 관측량 추정의 상대오차 ∝ 1/(<sgn> √N_samp) 가 어떻게 폭발하는지 모사한다.
def avg_sign(beta, N, df=0.05):
    """평균 부호 = exp(-β N Δf) (자유에너지 밀도 차 df>0)"""
    return np.exp(-beta * N * df)

def estimate_with_qmc(beta, N, n_samp=200000, df=0.05):
    """|W| 표본추출 + 부호를 관측량에 넣는 QMC 모사.
       각 표본의 부호를 +1(확률 p_plus)/-1 로 뽑아 <sgn> 추정."""
    s = avg_sign(beta, N, df)
    p_plus = (1 + s) / 2                     # <sgn>=2p_plus-1 가 되도록
    signs = np.where(rng.random(n_samp) < p_plus, 1.0, -1.0)
    sgn_mean = signs.mean()
    sgn_err = signs.std() / np.sqrt(n_samp)  # 분모 <sgn>의 통계오차
    return s, sgn_mean, sgn_err

print(" β   N   <sgn>(이론)   <sgn>(QMC)±err     분모 상대오차")
for beta in [1.0, 2.0, 4.0]:
    for N in [8, 16, 32, 64]:
        s_th, s_qmc, s_err = estimate_with_qmc(beta, N)
        rel = s_err / max(abs(s_qmc), 1e-12)
        flag = "  ← 신호 소멸!" if s_th < 1e-3 else ""
        print(f"{beta:.1f}  {N:3d}   {s_th:.2e}    {s_qmc:+.2e}±{s_err:.1e}   {rel:6.2f}{flag}")
# 핵심: β·N 이 커지면 <sgn>→0, 분모(평균 부호)의 상대오차가 발산.
# 정해진 정밀도를 위한 표본 수가 ∝ exp(2βNΔf) 로 폭발 → 부호 문제.
```

이 코드는 평균 부호 $\langle\mathrm{sgn}\rangle = e^{-\beta N\Delta f}$ 가 $\beta\cdot N$ 과 함께 지수적으로 0에 가까워져, QMC 분모의 상대오차가 발산하는 것을 보인다. 정해진 정밀도를 위한 표본 수가 $\propto e^{2\beta N\Delta f}$ 로 폭발하는 것이 부호 문제의 본질이다.

### 실험 2 — 얽힘 장벽: 면적 법칙과 결합차원

1D(면적 법칙, $S\sim$상수)와 2D(경계 법칙, $S\sim L$)에서 얽힘 엔트로피가 다르게 자라, 텐서망에 필요한 결합차원 $\chi\gtrsim e^S$ 가 2D에서 지수적으로 커짐을 보인다.

```python
import numpy as np

# 1D 자유 페르미온 사슬에서 부분계 얽힘 엔트로피를 상관행렬로 정확 계산.
# (Peschel 방법: 얽힘 S 는 상관행렬 C 의 고윳값으로부터)
def entanglement_1d(L_total, L_sub, filling=0.5):
    """반채움 1D 긴밀결합 사슬, 길이 L_sub 부분계의 얽힘 엔트로피"""
    # 단일입자 고유함수 (열린 경계): 정현파
    n = np.arange(1, L_total + 1)
    modes = np.arange(1, L_total + 1)
    psi = np.sqrt(2 / (L_total + 1)) * np.sin(np.outer(n, modes) * np.pi / (L_total + 1))
    Nocc = int(filling * L_total)
    occ = psi[:, :Nocc]                       # 채워진 모드
    C = occ @ occ.T                           # 상관행렬 <c†_i c_j>
    Csub = C[:L_sub, :L_sub]                  # 부분계로 제한
    nu = np.linalg.eigvalsh(Csub)
    nu = np.clip(nu, 1e-12, 1 - 1e-12)
    return -np.sum(nu * np.log(nu) + (1 - nu) * np.log(1 - nu))

print("1D 임계 사슬: S(L_sub) ~ (c/3) ln L_sub  (로그, 거의 면적 법칙)")
for Ls in [4, 8, 16, 32, 64]:
    S = entanglement_1d(256, Ls)
    print(f"  L_sub={Ls:3d}   S={S:.3f}   필요 χ≳e^S={np.exp(S):.1f}")

# 2D 면적 법칙 모사: 경계 길이에 비례 S ~ α L  → χ ≳ e^{αL} 지수 폭발
print("\n2D 면적 법칙(모사): S ~ α·L  → 결합차원 χ ≳ e^{αL}")
alpha = 0.3
for L in [4, 8, 16, 32]:
    S2d = alpha * L
    print(f"  L={L:3d}   S≈{S2d:.1f}   필요 χ≳e^S={np.exp(S2d):.2e}")
# 핵심: 1D 임계계는 S가 ln L 로 느리게 자라 χ 가 다항적.
# 2D는 S~L 로 χ~e^L 지수 폭발 → DMRG/텐서망의 얽힘 장벽.
```

이 코드는 1D 자유 페르미온 사슬의 얽힘 엔트로피를 상관행렬로 정확히 계산해 임계계의 $S\sim(c/3)\ln L$ 로그 증가를 확인하고, 2D 면적 법칙 $S\sim\alpha L$ 에서 필요 결합차원 $\chi\gtrsim e^{\alpha L}$ 이 지수 폭발함을 대조한다 — 텐서망 얽힘 장벽의 정량적 그림.

## 🧪 실험·관측 증거

- **Troyer–Wiese (*PRL* 2005)**: 일반 부호 문제를 다항시간에 푸는 것이 NP-난해임을 증명 — 부호 문제가 원리적 장벽임의 엄밀한 근거.
- **냉원자 허바드 양자 시뮬레이션 (Ch6-03, Mazurenko et al. 2017; Hartke et al. 2023)**: 고전 컴퓨터가 막히는 도핑 영역의 허바드 상관을 *물리적으로* 측정 — 부호 문제 우회.
- **DMRG 2D 원통 계산 (Stoudenmire–White 등)**: 유한 폭 원통에서 $t$-$J$ 의 줄무늬(stripe) 질서와 초전도 상관을 추출하나, 폭을 키울수록 결합차원이 폭발해 외삽에 큰 불확실성 — 얽힘 장벽의 실증.
- **DMFT 의 모트 전이 (Georges et al., *RMP* 1996)**: 무한차원 허바드의 모트 전이를 정량적으로 — 국소 상관은 풀지만 $d$-파 짝짓기는 군집 확장 필요.

## 🔬 경계 — 미해결의 정직한 지도

- **부호 문제는 완전 일반 해법이 없다**(NP-난해). 모형·기저별 완화는 있으나 도핑된 허바드·좌절 자성은 여전히 막힘.
- **2D·동역학의 얽힘 장벽**: 텐서망이 2D 바닥상태·실시간 발전에서 지수 비용. 부분 우회(PEPS·원통)는 통제된 외삽이 어려움.
- **방법 간 불일치**: 같은 모형에 QMC·DMRG·DMFT·변분이 다른 답을 줄 수 있어 *어느 답이 맞는지* 판정 기준 자체가 미해결인 영역이 있다(예: 2D 허바드 바닥상태의 줄무늬 vs 균일 초전도 경쟁).
- **양자 시뮬레이터의 검증 문제**: 시뮬레이터가 "푼" 답을 *검증* 하려면 다시 고전 계산이 필요한 역설 — 신뢰 구축이 진행 중.

## 🧬 횡단 원리

- **얽힘이 계산 복잡도를 정한다(Ch5-04)** — 면적 법칙 얽힘이 적으면 효율적 표현 가능, 많으면 불가능. 얽힘 구조가 *물질의 상* 만이 아니라 *계산 가능성* 을 규정한다(→ quantum-information).
- **창발과 환원(Ch1, L6)** — 미시 법칙을 알아도 창발을 *계산* 으로 예측 못 하는 곳. More is Different이 "원리적 어려움" 으로 구체화(→ L6 emergence).
- **양자–고전 대응(Ch6-01)** — 부호 문제는 양자계를 고전 통계로 사상할 때 음수 가중치로 나타난다. 양자성이 고전 모사를 막는 정확한 지점(→ Ch6-01).

## 🪜 창발

강상관 다체의 계산 장벽은 위 레이어에서 *기회* 로 뒤집힌다. **L5 holographic-principle** 은 강상관 양자계를 한 차원 높은 *고전 중력* 으로 사상해 — 얽힘 장벽·부호 문제 *없이* — 수송·상도를 계산하려는 시도다(AdS/CFT). 얽힘이 기하가 되는 곳(Ryu–Takayanagi)에서 다체 얽힘 구조가 시공간으로 번역된다. **L6 emergence** 에서 이 최전선은 "창발을 계산하는 것의 한계" 그 자체 — 환원주의가 원리적으로 막히는 지도다. 검증: 홀로그래피·양자 시뮬레이터가 예측한 강상관 관측량(상도·수송계수)을 실험과 대조하고, 방법 간 수렴을 확인하는 것.

## 📐 예측·반증

- **부호 문제 스케일링**: $\langle\mathrm{sgn}\rangle\sim e^{-\beta N\Delta f}$ — 표본 비용이 $\beta\cdot N$ 에 지수적. 다항 비용 해법이 일반 모형에서 나오면 NP=P 함의(거의 불가능).
- **얽힘–복잡도 관계**: 면적 법칙 위반(임계·2D)에서 $\chi\sim e^{S}$. 다항 $\chi$ 로 2D 임계계가 풀리면 텐서망 이론 수정.
- **방법 수렴 검증**: 부호 문제 없는 모형(반채움 허바드)에서 QMC·DMRG·DMFT가 같은 답으로 수렴해야 함 — 불일치는 구현 오류나 외삽 실패.
- **양자 우위 검증**: 양자 시뮬레이터가 고전 한계를 넘는 시스템 크기·시간에서 검증 가능한 예측(예: 허바드 $d$-파 상관)을 내면 우위 확립.

## 🤔 다음 질문

강상관의 최전선은 *계산* 의 한계지만, 그 너머에는 새 *언어* 가 있다. 만약 다체 얽힘을 직접 기하로 번역할 수 있다면(홀로그래피), 부호 문제·얽힘 장벽을 우회할 수 있을까? Chapter 6은 여기서 끝나고, 이제 Chapter 7로 넘어가 — 지금까지 유도한 모든 창발(밴드·천 수·BCS 갭·정확 대각화)을 *직접 계산* 으로 종합한다. 강상관을 작은 격자에서나마 손에 쥐어 본다.

---

🧩 **Principle** — 강상관 다체가 어려운 것은 계산량이 아니라 원리적 장벽이다: 페르미온 부호가 몬테카를로 가중치를 음수로 만들어 평균 부호가 $e^{-\beta N\Delta f}$ 로 소멸하고, 면적 법칙을 넘는 얽힘이 텐서망 결합차원 $\chi\sim e^S$ 를 지수 폭발시킨다.
🔬 **Boundary** — 일반 부호 문제는 NP-난해라 다항 해법이 없고, 2D·실시간 텐서망은 얽힘 장벽에 막혀 방법 간 답이 갈리는 미해결 영역이 남는다.
🪜 **Emergence** — 위 레이어에서 홀로그래피가 강상관을 고전 중력으로 번역해 장벽을 우회하려 하고, 양자 시뮬레이터가 자연에 계산을 떠넘기며, 상도·수송 대조로 검증된다.

## 📝 연습문제

**1. (기초)** 평균 부호 $\langle\mathrm{sgn}\rangle = e^{-\beta N\Delta f}$ 일 때, 정해진 상대오차를 얻기 위한 표본 수가 시스템 크기 $N$ 에 지수적임을 보여라.

<details><summary>해설</summary>

QMC 추정 $\langle O\rangle = \langle O\,\mathrm{sgn}\rangle_{|W|}/\langle\mathrm{sgn}\rangle_{|W|}$. 분모 $\langle\mathrm{sgn}\rangle$ 의 통계오차는 $\sigma_{\mathrm{sgn}}/\sqrt{N_s}$, 여기서 $\sigma_{\mathrm{sgn}}\sim O(1)$(부호는 $\pm1$). 상대오차 $= \sigma_{\mathrm{sgn}}/(\sqrt{N_s}\,\langle\mathrm{sgn}\rangle) \sim 1/(\sqrt{N_s}\,e^{-\beta N\Delta f}) = e^{\beta N\Delta f}/\sqrt{N_s}$. 이를 고정값 $\epsilon$ 으로 만들려면 $N_s \sim e^{2\beta N\Delta f}/\epsilon^2$ — 시스템 크기 $N$ 과 역온도 $\beta$ 에 *지수적*. 따라서 저온·대형 계에서 QMC가 실패. $\square$
</details>

**2. (심화)** 1D 갭 있는 바닥상태가 면적 법칙 $S\sim$상수를 따를 때 DMRG가 효율적인 이유를, MPS 결합차원 $\chi$ 와 $S\leq\ln\chi$ 로 설명하라. 2D에서 왜 무너지는가?

<details><summary>해설</summary>

MPS의 절단면 얽힘은 $S\leq\ln\chi$ 로 제한된다(슈미트 계수가 $\chi$ 개). 1D 갭 있는 상은 면적 법칙으로 $S\sim$상수($L$ 무관) → $\chi\sim e^{S}=$상수면 충분 → 변분 매개변수 $\sim N\chi^2$ 가 다항적 → DMRG 효율적. 실제로 1D 갭 있는 계는 $\chi\sim$수백이면 기계정밀도. 2D: $L\times L$ 격자를 MPS로 펴면(1D 사상) 절단면을 가로지르는 얽힘이 2D 면적 법칙으로 $S\sim L$(경계 길이). 그러면 $\chi\gtrsim e^{cL}$ — 폭에 지수적. 연산 $\sim\chi^3\sim e^{3cL}$ 폭발. 임계 1D도 $S\sim(c/6)\ln L$ 로 약하게 자라 $\chi$ 가 다항적으로 커지지만(여전히 다룰만), 2D는 본질적으로 막힌다. 면적 법칙의 *차원 의존성* 이 가능/불가능을 가른다. $\square$
</details>

**3. (논문 비평)** 양자 시뮬레이터가 "고전적으로 풀 수 없는 허바드 모형을 풀었다" 고 주장할 때, 이 주장을 진짜 "양자 우위" 로 받아들이기 전에 무엇을 검증해야 하는가?

<details><summary>해설</summary>

(1) **검증 역설**: 시뮬레이터의 답이 옳은지 확인하려면 독립적 계산이 필요한데, 고전적으로 못 푸는 영역이라면 검증 수단이 없다. 해법: 고전적으로 풀리는 영역(반채움·1D·작은 계)에서 먼저 시뮬레이터를 *교정* 하고, 그 신뢰를 미지 영역으로 외삽. (2) **시뮬레이터 불완전성**: 냉원자는 유한 온도($T\sim J$)·가둠 퍼텐셜·가열·유한 수명을 가져 *이상적* 허바드가 아님 — "푼" 모형이 목표 모형과 같은지 점검. (3) **관측량의 한계**: 시뮬레이터가 측정하는 양(밀도·짧은 거리 상관)이 미해결 질문($d$-파 짝짓기·$T_c$)에 직접 답하는지. (4) **재현성**: 다른 플랫폼(이온 트랩·초전도 큐비트)이나 다른 그룹이 같은 답을 재현하는지. 결론: "양자 우위" 주장은 *교정된 신뢰 + 모형 충실성 + 재현성* 의 삼박자가 갖춰져야 하며, 현재는 부분 교정 단계 — 인상적이나 결정적 우위는 아직. $\square$
</details>
