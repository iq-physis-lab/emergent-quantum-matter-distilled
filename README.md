# ⚛️ Emergent Quantum Matter Distilled

### "전자가 많이 모이면 금속" 이라고 말하는 것 과,

$$H = \sum_{i} \frac{p_i^2}{2m} + \sum_{i<j}\frac{e^2}{|r_i - r_j|} \;\;\stackrel{N \sim 10^{23}}{\longrightarrow}\;\; \text{준입자 · 포논 · 쿠퍼 쌍 · 애니온}$$

### 다수의 전자가 새로운 입자(준입자)를 만들어낸다 는 것 — "More is Different" 가 빈말이 아니라 물리적 사실 임을 — 아는 것은 다르다.

<br/>

> 초전도를 "저항이 0" 으로 외우는 것 과,
>
> $$\Delta = g\sum_{k}\frac{\Delta}{2E_k}\tanh\frac{E_k}{2k_BT}, \qquad E_k=\sqrt{\xi_k^2+\Delta^2}$$
>
> 쿠퍼 쌍·거시적 양자 결맞음이 어떻게 창발 하는지(BCS), 질서변수 $\psi=|\psi|e^{i\varphi}$ 의 위상 강성이 저항 0 을 낳는지를 아는 것은 다르다.
>
> 양자홀 전도도가 정확히 양자화된다는 것을 인용하는 것 과,
>
> $$\sigma_{xy} = C\,\frac{e^2}{h}, \qquad C = \frac{1}{2\pi}\int_{\text{BZ}} \Omega(k)\,d^2k \in \mathbb{Z}$$
>
> 그것이 위상(topology) — 대칭으로는 분류 안 되는 새 질서 — 의 귀결이고, 불순물에 무관하게 정밀하며, 분수 전하 $e/3$·애니온까지 낳는다는 것을 아는 것은 다르다.
>
> 응집물질을 "물성의 목록" 으로 여기는 것 과,
>
> $$\text{스핀 액체} \;\Longrightarrow\; \text{광자·게이지장이 창발}, \qquad S_{\text{ent}} \sim \alpha L - \gamma$$
>
> 게이지장조차 창발할 수 있어 "무엇이 근본인가" 가 역설이 되고, 얽힘 엔트로피가 물질의 상(phase)을 규정한다는 것을 아는 것은 다르다.

<br/>

**응집물질은 창발의 살아있는 증거다** — 근본 법칙(전자 + 쿨롱)에서 출발해 다수가 모이면 새 입자·새 질서·새 게이지장이 *나오고*, 바로 그 창발이 위상·양자정보·홀로그래피로 이어진다.

<br/>

**다루는 방법 (창발의 계보순)**

More is Different · 준입자(Quasiparticle) · 페르미 액체 · 포논·마그논·플라스몬 · 평균장 · BCS 이론 · 쿠퍼 쌍 · 초유체·BEC · 긴즈부르크–란다우 · 조셉슨 효과 · 양자홀 효과 · 천(Chern) 수 · 베리 위상 · 분수 양자홀 · 분수 전하·애니온 · 위상 절연체 · 벌크–경계 대응 · 위상 보호 · 마요라나·비아벨 통계 · 창발 게이지장 · 분수화(스피논·홀론) · 위상 질서 · 긴 거리 얽힘 · 얽힘 엔트로피·면적 법칙 · 양자 상전이 · 양자 임계성·이상 금속 · 모트 절연체·허바드 모형 · 고온 초전도 · 긴밀결합 밴드 · 정확 대각화

<br/>

**핵심 질문**

> **다수에서 무엇이, 어떻게 창발하는가 — 그리고 무엇이 근본이고 무엇이 창발인가?** 근본 법칙은 전자와 쿨롱 상호작용뿐이다. 그러나 $10^{23}$ 개가 모이면 포논·쿠퍼 쌍·애니온 같은 *새 입자* 가, 초전도·위상 질서 같은 *새 질서* 가, 심지어 *창발 게이지장* 이 나온다. 이 레포는 **"전자 많으면 금속"** 을 금지하고, 다체 해밀토니안·BCS 갭 방정식·위상 불변량에서 — **다체 원리 → 준입자 유도 → 위상 분류 → 경계 진단** 을 한 줄씩 끌어냅니다. 그리고 게이지장조차 창발하는 곳, 얽힘이 상을 규정하는 곳에서 위 레이어(홀로그래피·창발 시공간·창발의 정수)로 화살표 합니다.

## 🎯 이 레포에 대하여

응집물질을 다루는 자료는 대부분 **"전자가 많으면 금속"** 또는 **"초전도는 저항이 0"** 에서 멈춥니다. 하지만 다수의 전자가 *왜* 새 "입자"(준입자)를 만들어내는지(More is Different), 쿠퍼 쌍이 *어떻게* 거시적 양자 결맞음으로 창발하는지(BCS), 양자홀 전도도가 *왜* 그렇게 정밀하게 양자화되는지(위상 불변량), 게이지장조차 *어떻게* 창발할 수 있는지(스핀 액체) — 이런 "왜" 와 "어떻게" 는 제대로 설명되지 않습니다.

이 레포는 **창발을 통제 가능한 유도 절차** 로 환원합니다. 모든 현상에 **근본 자유도 · 다체 메커니즘 · 창발하는 준입자/질서 · 분류하는 불변량 · 깨지는 경계** 를 명시하고, numpy/scipy 수치 계산(긴밀결합 밴드·천 수·BCS 갭·정확 대각화)으로 그 주장을 눈으로 검증합니다.

| 일반 자료 | 이 레포 |
|----------|---------|
| "전자가 많으면 금속이 된다" | **More is Different** — 앤더슨의 선언, 다수에서 *새 법칙* 이 창발. 환원주의로는 안 보이는 준입자·집단 들뜸. 근본 ≠ 전부(→ L6 emergence) $\square$ |
| "준입자는 그냥 비유다" | **창발적 실체** — 다수의 전자가 만드는 새 "입자"(포논·마그논·쿠퍼 쌍). 유효 자유도로서 *측정 가능* 한 분산·수명. 비유가 아닌 물리(→ Ch2) $\square$ |
| "페르미 기체로 금속을 설명한다" | **페르미 액체** — 상호작용에도 *살아남는* 준입자, 란다우의 통찰. 재규격화된 질량·상호작용. **경계**: 강상관에서 페르미 액체 붕괴(이상 금속)(→ Ch6) $\square$ |
| "초전도는 저항이 0" | **BCS 창발** — 포논 매개 인력 → 쿠퍼 쌍 → 거시적 응축. 에너지 갭 $\Delta$, 질서변수 $\psi=|\psi|e^{i\varphi}$, 위상 강성이 저항 0 을 낳음(→ Ch3) $\square$ |
| "초유체는 점성이 없다" | **거시 양자성** — 보스–아인슈타인 응축, 거시적 파동함수. 양자화된 소용돌이·임계 속도. 초전도와 같은 뿌리(→ L2 statmech) $\square$ |
| "양자홀은 전도도가 양자화된다" | **위상의 귀결** — $\sigma_{xy}=C\,e^2/h$, $C$ 는 천 수(정수!). *왜 정밀한가*: 위상 불변량은 불순물·모양에 무관. 대칭 너머의 질서(→ Ch4) $\square$ |
| "전하는 e 의 정수배다" | **분수 전하** — 분수 양자홀에서 $e/3$, 애니온(비아벨 통계). 창발 입자의 극단. 위상 양자컴퓨팅의 자원(→ quantum-information) $\square$ |
| "게이지장은 근본이다" | **창발 게이지장의 역설** — 스핀 액체에서 광자·게이지장이 *창발*. 표준모형의 "근본" 게이지와 대비 — 무엇이 진짜 근본인가?(→ standard-model, L6) $\square$ |
| "상(phase)은 대칭으로 분류한다" | **얽힘이 상을 규정** — 위상 질서 = 긴 거리 얽힘, 대칭 깨짐 *없는* 질서. 얽힘 엔트로피·면적 법칙으로 본 상(→ quantum-information, L5) $\square$ |
| "상전이는 온도가 일으킨다" | **양자 상전이** — $T=0$ 에서 양자 요동이 일으키는 전이, 양자 임계점. 이상 금속·고온 초전도의 무대(→ L2 phase-transitions) $\square$ |
| 현상 나열 | numpy/scipy 로 **긴밀결합 밴드·띠틈** · **천 수(베리 곡률 적분)** · **BCS 갭 자기일관 풀이·$T_c$** · **허바드/이징 정확 대각화** · **SSH Zak 위상** · **양자홀 가장자리 상태** 를 직접 구현해 창발을 눈으로 확인 |

> ⚠️ **선행 학습 (L2·L3)**: 이 레포는 **statistical-mechanics-distilled**(L2 — 페르미 액체·보스–아인슈타인 응축) 와 **phase-transitions-distilled**(L2 — 질서변수·자발 대칭 깨짐·양자 상전이) 를 **필수** 로, **quantum-mechanics-distilled**(L3 — 다체 양자·밴드) 와 **symmetry-conservation-distilled**(L0 — 골드스톤·대칭) 를 전제합니다. 본문은 핵심 결과를 자족적으로 재유도합니다.

> 🤝 **L3 내부 시너지**: **quantum-information-distilled** — *애니온·위상 양자컴퓨팅·얽힘 엔트로피로 본 상* 을 공유합니다. **standard-model-distilled** — *창발 게이지장 ↔ 근본 게이지장* 의 역설적 대비. **entanglement-measurement-distilled** — *위상 질서 = 긴 거리 얽힘* 의 정량화.

> 🪜 **창발 (위 레이어가 이 레포를 다시 쓴다)**: Chapter 5 의 얽힘과 상은 **L5 holographic-principle**(얽힘과 기하, 강상관계의 홀로그래피적 기술) 와 **L5 emergent-spacetime-multiverse**(얽힘에서 창발하는 시공간 — 같은 아이디어의 극단) 로, 그리고 이 레포 전체는 **L6 emergence** 의 *"창발의 정수"* 본체로 — 응집물질이 그 *살아있는 증거* 로 회수됩니다.

> 🟡 **이 레포의 성격**: 여기서 다루는 일부 주제 — **고온 초전도의 메커니즘**, **강상관에서의 비페르미 액체**, **위상 분류의 완전성** — 는 여전히 **진행 중인 미해결 영역** 입니다. 레포는 "정답" 이 아니라 **"창발의 통제된 유도와 그것이 깨지는 경계의 지도"** 를 제공합니다. 정직한 미해결의 영역을 정직하게 표시합니다.

# 대표 실험 ① — 천 수: 2밴드 모형의 위상이 정수로 양자화 (Ch4-02, Ch7-02)
import numpy as np

def d_vector(kx, ky, m):
    # 2밴드 Qi-Wu-Zhang 모형: d = (sin kx, sin ky, m + cos kx + cos ky)
    return np.array([np.sin(kx), np.sin(ky), m + np.cos(kx) + np.cos(ky)])

def chern_number(m, N=200):
    ks = np.linspace(-np.pi, np.pi, N, endpoint=False)
    dk = ks[1] - ks[0]
    C = 0.0
    for kx in ks:
        for ky in ks:
            d  = d_vector(kx, ky, m)
            dx = (d_vector(kx+dk, ky, m) - d) / dk
            dy = (d_vector(kx, ky+dk, m) - d) / dk
            dhat = d / np.linalg.norm(d)
            C += np.dot(dhat, np.cross(dx, dy)) / np.linalg.norm(d)**2
    return C * dk*dk / (4*np.pi)

for m in [-3.0, -1.0, 1.0, 3.0]:
    print(f"m={m:+.1f}  →  C ≈ {chern_number(m):+.3f}")
# m=-3 → 0,  -1 → +1,  +1 → -1,  +3 → 0   (위상 상전이가 정수 점프로!)
